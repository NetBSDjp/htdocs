building ifconfig(1/3)   ifconfigの作成(1/3)


Make sure you have installed BIND8 already from pkgsrc.
あなたが pkgsrc から既に BIND8 をインストールしたことを確かめて下さい。

Checkout Commands tree from darwin repository.
darwin リポジトリからのコマンドツリーのチェックアウト。

 % cvs -d :pserver:yourid@anoncvs.opensource.apple.com:/cvs/Darwin checkout Commands
