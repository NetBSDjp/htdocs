Building IPv6-ready SSH(2/2)   IPv6 対応 SSH の作成(2/2)


If you would like to start IPv6-ready sshd at startup, modify /System/Library/StartupItems/SSH/SSH.
もしあなたが起動時に IPv6 対応 sshd を起動したければ、/System/Library/StartupItems/SSH/SSH を修正して下さい。

(訳注: Apple menu - System Preference... - Sharing の Application タブの Allow remote login チェックボックスの動作に注意。IPv6 対応 sshd だけ SSH6 にして別にすることで Apple の updater との親和性を高め不具合を減らすことができるかも知れない。)
