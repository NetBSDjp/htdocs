Welcome to the Japan NetBSD Users' Group

``Of course it runs NetBSD.''


日本NetBSDユーザーグループ(Japan NetBSD Users' Group, JNUG)は、 日本でのNetBSDユーザーの情報交換を目的として設立されました。

JNUG ニュース (過去のニュース)

2002/06/27
7月3日に NetWorld+Interop 2002 Tokyo にて、BOF 〜BSDなひととき〜が開催されます。
2002/06/27
7月3日(水)に、関東にある4BUG(BSD Users Group)合同で 2002年春のイベント (関東BUG 合同合宿) を開催します。
2002/04/22
5月18日(土),19(日)に、関東にある4BUG(BSD Users Group)合同で 2002年春のイベント (関東BUG 合同合宿) を開催します。
2002/04/13
5月18日(土)にJNUG第四回定期総会とNetBSD BOF 2002が開催されます。
2002/3/13
3月31日(日)に、緒方様が pkgsrc BOF を開催します。
[UPDATE 2002/05/01]当日の 資料を追加しました。
2002/3/13
anoncvs.jp.netbsd.org と cvsweb.netbsd.org の CVSリポジトリが復旧 しました。 rsync の LP64 関係の不具合が原因でした。長期間にわたり御迷惑 をおかけして申し訳ありませんでした。
