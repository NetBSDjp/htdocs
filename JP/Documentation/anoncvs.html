Anonymous CVS サーバー

Anonymous CVS

JP.NetBSD.ORGでは、NetBSDの開発用 CVS リポジトリーと JP.NetBSD.ORGローカル(Webコンテンツや開発プロジェクト用など) のCVS リポジトリーを、読み取り専用ではありますが、 誰でも自由に利用できるように Anonymous CVS サービスを行っています。

Anonymous CVSに関する情報
  • CVS Information(CVS情報 and ドキュメント)
  • FreeBSDハンドブック
  • OpenBSD AnonCVS
  • CVS Access to cvs.samba.org

使い方

準備
  1. cvsをインストールします。 NetBSD cvs packageを使用するのが簡単です。

  2. ログインします。 
    cvs -d :pserver:anoncvs@sup.jp.netbsd.org:CVSROOT login

    CVSROOT は以下のどちらかを指定します。
    NetBSD.ORG /cvs/cvsroot
    JP.NetBSD.ORG /cvs/jproot
    例: NetBSD.ORGリポジトリーの場合
    cvs -d :pserver:anoncvs@sup.jp.netbsd.org:/cvs/cvsroot login

  3. CVS passwordの入力を求められるので、

    anoncvs

    と入力します。

checkout
NetBSD-currentの最新ソースコードをcheckoutするには、 
cvs -d :pserver:anoncvs@sup.jp.netbsd.org:/cvs/cvsroot co src
です。

update
check outしたmoduleを最新のものに同期させるには、 checkoutしたディレクトリーで、 
cvs update -P -d
です。

CVS Web インターフェース

CVS リポジトリーは Web インターフェース からも閲覧可能です。
