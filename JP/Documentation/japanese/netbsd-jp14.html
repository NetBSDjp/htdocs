GhostView 1.5のインストール

GhostViewは、GhostScript のフロントエンドとして動作する PostScript Vieer です。拡大、縮小や、ページの切替えなど をマウス操作で簡単に行なうことができます。

ソースの修正

ソースを展開します。 以下の場所などにあります。

ftp://ftp.tut.ac.jp/GNU/prep/ghostview-1.5.tar.gz

misc.c

64行目の 

extern char *sys_errlist[];
を削除します

SelFile.c

49行目の 

extern char *sys_errlist[];
を削除します

コンパイル/インストール

% xmkmf
% make all
% su
# make install
# make install.man

GhostViewの使用方法

% ghostview ファイル名
として利用します。ファイル名を省略すると、選択のメニューが出ます。
