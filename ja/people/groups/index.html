NetBSD People: NetBSD プロジェクト内の別のグループ

NetBSD プロジェクト内の別のグループ

このページでは NetBSD 内部の別のグループと、彼らの担当分野、 連絡先を列挙しています。これは、下記への追加情報です: ほとんどの NetBSD 開発者は、どのグループにも属していませんが、それでも彼らは プロジェクトに寄与し、実際に貢献しています。この人達がいなかったら NetBSD は存在しなかったでしょう。開発者の一般的なグループには以下のものがあります。

  • NetBSD 開発者
  • NetBSD コアグループ
  • NetBSD Foundation

このほか、 NetBSD プロジェクトには以下のグループがあります。

  • Accounts - accounts@NetBSD.org
  • Admin - admin@NetBSD.org
  • Announce - announce@NetBSD.org
  • Mirrors - mirrors@NetBSD.org
  • Packages - packages@NetBSD.org
  • pkgsrc Releng - releng-pkgsrc@NetBSD.org
  • pkgsrc Security Team - pkgsrc-security@NetBSD.org
  • Releng - releng@NetBSD.org
  • Security Alert - security-alert@NetBSD.org
  • Security-Officer - security-officer@NetBSD.org
  • Security Team - security-team@NetBSD.org
  • WWW - www@NetBSD.org

グループ

Accounts - accounts@NetBSD.org (top)

NetBSD プロジェクトに加わった開発者のための、 プロジェクトホスト開発者アカウント作成を担当します。

メンバー: NetBSD Admin グループ

Admin - admin@NetBSD.org (top)

NetBSD プロジェクトのホストの変更を担当します。 アカウント関連の問題も、このグループが適切な報告先です。

メンバー: Tracy Di Marco White, Phil Nelson, Jonathan Perkin, Jeff Rizzo, Jan Schaumann, Silke Scheler, Amitai Schlair, Thor Lancelot Simon, Noriyuki Soda, T K Spindler, Jason Thorpe, Krister Walfridsson, S.P.Zeidler, Christos Zoulas.

Announce - announce@NetBSD.org (top)

<netbsd-announce@NetBSD.org> への投稿のモデレートを行ないます。 現在は Herb Peyerl がこの仕事を担当しています。

Mirrors - mirrors@NetBSD.org (top)

ミラー運営組織の要望や情報交換を処理するグループです。

メンバー: WWW.

pkgsrc Releng - releng-pkgsrc@NetBSD.org (top)

pkgsrc のリリースエンジニアリンググループです。 安定版の枝の管理、pullup 手続、凍結期間の策定をおこないます。

メンバー: Geert Hendrickx, Tyler R. Retzlaff, Matthias Scheler, Lubomir Sedlacik, S.P.Zeidler.

pkgsrc Security Team - pkgsrc-security@NetBSD.org (top)

pkgsrc のセキュリティーに関する事項の処理を担当します。

メンバー: Mihai Chelaru, Alistair G. Crooks, Ulrich Habel, Thomas Klausner, Guillaume Lasmayous, Tonnerre Lombard, Tobias Nygren, OBATA Akio, Fredrik Pettai, Adrian Portelli, Jörg Sonnenberger, Tim Zingelman.

Releng - releng@NetBSD.org (top)

リリースエンジニアリンググループ。リリース予定、方針、pullup を計画し 実行します。

メンバー: Stephen Borrill, Manuel Bouyer, David Brownlee, James Chacon, Julian Coleman, Alistair G. Crooks, Havard Eidnes, Jaime A Fournier, Liam J. Foy, John Heasley, Martin Husemann, Soren Jacobsen, Mark Kirby, Phil Nelson, Jeff Rizzo, SAITOH Masanobu.

Security Alert - security-alert@NetBSD.org (top)

NetBSD プロジェクトのセキュリティー問題を知らせるための緊急連絡アドレスです。 下記の security-officer ほどは「グループ」の体裁をとっていません。

メンバー: Matthew Green, David Maxwell, NetBSD コアグループ, NetBSD Security-Officer グループ.

Security-Officer - security-officer@NetBSD.org (top)

セキュリティー問題の解決と告知の扱いを担当します。

メンバー: Daniel Carosone, Alistair G. Crooks, Tracy Di Marco White, Tonnerre Lombard, Adrian Portelli.

Security Team - security-team@NetBSD.org (top)

セキュリティー問題の解決の手伝いと告知の準備を担当します。

Members: David Brownlee, Simon Burge, Bill Coldwell, Matthew Green, David A. Holland, Martin Husemann, David Maxwell, der Mouse, John Nemeth, John R. Shannon, Jörg Sonnenberger, Matt Thomas, Christos Zoulas, NetBSD Security-Officer

WWW - www@NetBSD.org (top)

NetBSD web サイトの内容の保守および追加を担当するグループです (開発者全員が htdocs のチェックアウトをしたり、追加すべき内容を追加することが 推奨されてはいますが) 。また、 web ページフォームからのフィードバックの反映や、 NetBSD ギャラリーへの投稿の処理もおこないます。

メンバー: David Brownlee, D'Arcy J.M. Cain, Sarah Cockburn, Tracy Di Marco White, Jun Ebihara, Erik Fair, Hubert Feyrer, Jay Fink, Michael Graff, Ulrich Habel, Klaus Heinz, Thomas Klausner, David Maxwell, Jared D. McNeill, Julio Merino, Phil Nelson, Mike Pelley, Herb Peyerl, Mindaugas Rasiukevicius, Jeremy C. Reed, Jan Schaumann, Reed Shadgett, Daniel Sieger, Noriyuki Soda, Mike M. Volokhov, Mark Weinem, Yuji Yamano.
