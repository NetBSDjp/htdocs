CD-ROMs with NetBSD

無料の ISO イメージは、ftp://ftp.NetBSD.org/pub/NetBSD/iso/ 以下および ミラーサイトにあります。

以下は、NetBSD の CD-ROM を入手する沢山の方法のうちのいくつかの一覧です。 もし、CD-ROM を販売しており、ここに掲載したい場合は、 私達に御連絡下さい。

最新の公式リリースはNetBSD 5.1 です。

ワールドワイド

5.0.2 - Linux System Labs (top)

Linux System Labs は、NetBSD 5.0.2 リリースの i386 および amd64 用 CD CD を販売しています。オーストラリアにある会社ですが、世界中に 発送してくれます。

4.0.1 - Linux Central (top)

Linux Central は NetBSD 4.0.1 の 3 枚組 CD を販売しています。

5.0.2 - CheepLinux (top)

CheepLinux では、 NetBSD 5.0.2 の CD を販売しています。 午後 3:30 までに届いた注文は、全世界に即日出荷されます。

5.0.2 - Linux-Distro.co.uk (top)

Linux Distro UK は、amd64, i386, macppc プラットフォーム用の NetBSD 5.0.2 を販売しています。イギリスにある会社ですが、 世界中に発送してくれます。

4.0, 3,1 - Mensys BV (top)

Mensys では NetBSD 4.0/i386 および 3.1/i386 の CD-ROM を販売しています。

3.1 - Maxtux.co.uk (top)

maxtux.co.uk では、 NetBSD 3.1 の i386 インストール CD と、 バイナリーパッケージ CD を販売しています。

5.0 - linuxcenter.ru (top)

LinuxCenter は、NetBSD 5.0 を販売しています。ロシアにある会社です。

Custom - Yong S. Yi: Release versions for all architectures (top)

Yong S. Yi <ysyi@async.org> も、カスタム CD を作成して います。CD は、指定されたアーキテクチャー(あるいはアーキテクチャーの組み合せ) のリリース、バイナリーパッケージ、ドキュメンテーションを含んでいます。 価格は US $6.50 で、アメリカ国内への送料も含まれています。

各国

カナダおよびアメリカ - 5.0.2 - OS Dics by Mail (全ポート) (top)

OS Discs By Mail では、 "Download-Burn-Mail" サービスを実施しています。 フリーなオペレーティングシステムのディスクをダウンロードして CD に焼いて送ってくれるサービスです。

カナダ - 5.0 - CheapBytes (top)

CheapBytes は、 リリースバイナリーとソースコードを収録した、i386 と amd64 の NetBSD 5.0 インストール CD を販売しています。

フランス - 5.0.1 - getlinux (top)

Getlinux は NetBSD 5.0.1 を販売しています。

ドイツ - 5.0 Linux2go (top)

ドイツが基盤の Linux2go は NetBSD 5.0 (i386, amd64) をドイツ国内向けに販売しています。

ドイツ/ヨーロッパ - 4.0 Lehmanns Bookshop (top)

ドイツが基盤 (出荷先はヨーロッパ地域) の書店 Lehmanns は、 NetBSD 4.0 CD 4 枚組を販売しています。

ドイツおよび EU - 4.0.1 - bsdiso - Software Service Ronald Brock (top)

ドイツの http://www.bsdiso.de/ では、 NetBSD/i386 その他のプラットフォームの 4.0.1 の CD を、 欧州連合向けに販売しています。

ドイツ - 5.0 - ixsoft Softwareentwicklung (top)

ixsoft Softwareentwicklung は、amd64 および i386向けの 5.0 DVD を販売しています。 詳細は 彼らのサイト を参照してください。

India - 4.0 - LinuxBazar (top)

http://www.linuxbazar.com/ は、 NetBSD/i386 4.0 を販売しています。バングラデシュ、 ブータン、ネパール、パキスタン、スリランカにも出荷します。

カザフスタン - 4.0 - linuxcenter.kz (top)

LinuxCenter.KZ では、NetBSD 3.x と 4.x を、カザフスタン共和国向けに販売しています。

ポーランド - 3.1 - e-systems (top)

e-systems では、 NetBSD/i386 の最新の安定版である 3.1 の CD-ROM を販売してます。

ロシア - 4.0.1 - OpenByte (top)

OpenByte.ru では、NetBSD 5.0.2 を、ロシア連邦向けに販売しています。

スペイン - 4.0.1 - Compralinux.com (top)

Compralinux.com では、 NetBSD 4.0.1 および旧バージョンの、複数アーキテクチャー対応および i386 専用の CD-ROM を、スペイン向けに販売しています。

アメリカ - 5.0 - CheapBytes (top)

CheapBytes の項を参照。

ウクライナ - 5.0.2 - Lafox.Net (amd64, i386) (top)

Lafox.Net は、 i386 および amd64 用の NetBSD 5.0.2 を販売しています。

ISO イメージのダウンロード

公式 NetBSD ISO イメージ (top)

NetBSD ISO ダウンロードサイト一覧に、 これらの公式 ISO イメージがダウンロードできるサイトの全世界の一覧 があります。

各ミラーサイトの iso イメージ用ディレクトリーには、ポート毎にひとつのイメージがあります。 CD や DVD からのブートに対応しているプラットフォーム上では、 このイメージからブート可能です。
