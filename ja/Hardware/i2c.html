Machine-independent I2C drivers

The I2C bus is a standard serial bus used as an interconnect between components on a device. This is a support bus, and does not export an interface to the user. However, having this generic I2C support can aid device driver writers, as they will not have to re-implement I2C support code if their device uses I2C.

Please note that these pages reflect the state of NetBSD-current. To check the supported hardware list for a specific release of NetBSD, check that release's installation notes.

Supported I2C devices

  • I2C EEPROMs (Microchip Technology 24C01A/02A/04A).
