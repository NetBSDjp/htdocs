機種に依存しない USB ドライバー

下記のデバイス(括弧内はドライバー名)が機種に依存しない USB ドライバーによってサポートされています。 リストにないデバイスは、各プラットフォームのページをチェックしてみてください。 そのデバイスのドライバーが、まだ機種独立になっていないのかもしれません。

別の可能性として、そのデバイスはリストにあるデバイスと互換かもしれませんし、 そのデバイスを見落していて、リストに載せ忘れているのかもしれません。 不明な点がある時は、 netbsd-help@NetBSD.org にメールしてください。

[訳注: netbsd-help は英語のメーリングリストです。日本語での問い合わせは、 ここを参照してください。]

このページはNetBSD-currentの現状を反映していることに注意してください。 特定のリリースでサポートされているハードウェアを調べるには、 そのリリースのインストレーション・ノートを参照してください。

サポートされている USB デバイス

  • USB ハブ (uhub)
  • USB キーボード (ukbd)
  • USB マウス (ums)
  • USB 汎用ヒューマン・インターフェース・デバイス (何でもあり) (uhid)
  • USB プリンター (ulpt)
  • USB モデム (umodem)
  • USB 大容量記憶デバイス、 例えば、 USB Zip ドライブ (umass)
  • USB オーディオ (uaudio)
  • USB midi (umidi)
  • USB スキャナー (uscanner, usscanner)
  • USB シリアルポート (uftdi, umct, uplcom)
  • USB イーサネットアダプター (aue, cue, kue)
  • USB-USB ケーブル (upl)
  • USB 汎用ドライバー (ugen)
  • Handspring Visor (uvisor)
  • Y@p Phone (uyap)
  • Diamond Rio 500, 600 (urio)
