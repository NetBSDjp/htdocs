機種に依存しない PCMCIA ドライバー

下記のデバイス(括弧内はドライバー名)が機種に依存しない PCMCIA ドライバーによってサポートされています。 リストにないデバイスは、各プラットフォームのページをチェックしてみてください。 そのデバイスのドライバーが、まだ機種独立になっていないのかもしれません。

別の可能性として、そのデバイスはリストにあるデバイスと互換かもしれませんし、 そのデバイスを見落していて、リストに載せ忘れているのかもしれません。 不明な点がある時は、 netbsd-help@NetBSD.ORG にメールしてください。

[訳注: netbsd-help は英語のメーリングリストです。日本語での問い合わせは、 ここを参照してください。]

このページはNetBSD-currentの現状を反映していることに注意してください。 特定のリリースでサポートされているハードウェアを調べるには、 そのリリースのインストレーション・ノートを参照してください。

サポートされている PCMCIA デバイス

  • マルチファンクションカード
    • Megahertz XJEM3336 イーサネット/モデム カード (mhzc, com, sm).
  • SCSI ホストアダプター (サポートされている SCSI ターゲットデバイス)
    • Adaptec AIC-6260 と AIC-6360 ベースのカード。 Adaptec APA-1460 を含む。(aic)
    • NCR 53C9x, Emulex ESP406, Qlogic FAS408 (esp).
  • ネットワーク・インターフェース
    • Aironet 4500/4800 及び、Cisco 340 シリーズ ワイヤレスネットワークカード (an)
    • BayStack 650 IEEE 802.11FH ワイヤレスネットワークカード (awi)
    • Netwave AirSurfer ワイヤレスネットワークカード (cnw)
    • 3Com 3c562, 3c574, 3c589 ファミリー (ep)
    • Fujitsu MB86960 ベースのカード、TDK CD021BX を含む。(mbe)
    • NE2000 互換品 (ne)
    • Raytheon Raylink / WebGear Aviator IEEE 802.11 2Mbps ワイヤレス (ray)
    • SMC91c9x ベースのカード、Megahertz X-Jack を含む。 (sm)
    • AT&T/Lucent WaveLan ワイヤレス PCMCIA カード (wi)
    • Intersil PRISM ワイヤレス PCMCIA カード (wi)
    • Xircom CreditCard PCMCIA イーサネット (xi)
    • IBM と 3Com の Tropic ベースのトークンリングカード (tr)
  • シリアルポート:
    • 16550 及び、16550と互換性のある UART とモデムカード。 モデム/イーサネット・コンビネーションカードを含む。(com)
    • マルチポート・シリアルカード。例えば、Socket Communications, Inc. の Dual Serial interface (comを子供として持つpcmcom)
  • ディスクとCD-ROMのコントローラー
    • MFM, ESDI, IDE, RLL のハードディスク・コントローラー。 Digital Mobile Media CD-ROM を含む。(wdc) (サポートされている ATA デバイス), (サポートされている ATAPI デバイス).
  • サウンド・ハードウェア
    • ESS 1688 AudioDrive (esl)
