機種に依存しない PCI ドライバー

下記のデバイス(括弧内はドライバー名)が機種に依存しない PCI ドライバーによってサポートされています。 リストにないデバイスは、各プラットフォームのページをチェックしてみてください。 そのデバイスのドライバーが、まだ機種独立になっていないのかもしれません。

別の可能性として、そのデバイスはリストにあるデバイスと互換かもしれませんし、 そのデバイスを見落していて、リストに載せ忘れているのかもしれません。 不明な点がある時は、 netbsd-help@NetBSD.ORG にメールしてください。

[訳注: netbsd-help は英語のメーリングリストです。日本語での問い合わせは、 ここを参照してください。]

サポートされている PCI デバイス

  • RAID コントローラー
    • DPT SCSI SmartCache III/IV と SmartRAID III/IV (dpt).
  • SCSI ホストアダプター (サポートされているSCSI ターゲットデバイス)
    • Adaptec AHA-2940, AHA-2940U, AHA-2940AU, AHA-2944, AHA-2944U, AHA-3940, AHA-3940U, AIC-7850, AIC-7855, AIC-7860, AIC-7870, AIC-7880 (ahc)
      注意: AIC-7890はまだサポートされていません。
    • Advanced Micro Devices Am53c974 PCscsi-PCI と、このチップを使っているボード。 Tekram DC-390 を含む。 (pcscp).
    • AdvanSys 1200[A,B], 9xx[U,UA] (adv)
    • AdvanSys 9xxUW (adw)
    • BusLogic 9xx (bha)
    • Qlogic ISP-1020, ISP-1040, ISP-1080 (LVD), 及び ISP-2100 (FibreChannel) (isp)
    • NCR/Symbios Logic 53c810, 53c810A, 53c815, 53c820, 53c825, 53c825A, 53c860, 53c875, 53c875J, 53c885, 53c895, 53c896 (ncr)
  • ネットワーク・インターフェース
    • Digital 21040, 21041, 21140, 21140A, 21142, 21143 ベースのカード。 Digital DE425, SMC Etherpower, Adaptec/Cogent マルチポート、 ZNYX マルチポート、 Bay Networks NetGearを含む。 (de)
    • Efficient Networks EN-155 と Adaptec AIC-590x ATM インターフェース (en)
    • 3Com 3c90x[B] (ex)
    • 3Com 3c59x (ep)
    • SMC EPIC/100 Fast Ethernet。 SMC Etherpower-II を含む。(epic)
    • Essential Communications Hippi (800 Mbit/s) (esh)
    • Digital DEFPA SAS と DAS FDDI カード (fpa)
    • Intel EtherExpress Pro 10/100B/100+ (fxp)
    • PCnet-PCI (le)
    • LAN Media Corporation DS3 インターフェース (lmc)
    • NE2000 互換品。 Realtek 8029, Winbond 89C940F, VIA Technologies VT86C926, Surecom NE-34, NetVin 5000, 及び、Compex, ProLAN, と KTI のモデルを含む。 (ne)
    • RISCom N2 同期インターフェース (ntwoを子供として持つntwoc)
    • Silicon Integrated Systems SiS 900 ベースの 10/100 イーサネットカード (sip)
    • Texas Instruments ThunderLAN ベースのカード。 Compaq Netelligent 10/100 TX, Compaq Netelligent 10 T, Compaq Integrated NetFlex 3/P, Compaq ProLiant Integrated Netelligent 10/100 TX, Compaq Dual Port Netelligent 10/100 TX, Compaq Deskpro 4000 5233MMX Ethernet, Compaq NetFlex 3/P w/ BNC, Compaq NetFlex 3/P, Texas Instruments TravelMate 5000 ドッキングステーションのイーサネットを含む。 (tl)
    • VIA VT3043 (Rhine) と VIA VT86C100A (Rhine-II) ベースの 10/100 イーサネットカード。 D-Link DFE530TX を含む。(vr)
  • 通信コントローラー:
    • PCI 「汎用」 通信カード (puc)。 シリアル (com) 及び、パラレル (lpt) ポートを提供します。 このドライバーは以下のボードをサポートします。
      • Dolphin Peripherals 4014 (パラレル×2)
      • Dolphin Peripherals 4035 (シリアル×2)
      • SIIG Cyber 2P1S PCI (パラレル×2、 シリアル×1)
      • SIIG Cyber 2S1P PCI (シリアル×2、パラレル×1)
      • SIIG Cyber 4S PCI (シリアル×4)
      • SIIG Cyber I/O PCI (シリアル×1、 パラレル×1)
      • SIIG Cyber Parallel Dual PCI (パラレル×2)
      • SIIG Cyber Parallel PCI (パラレル×1)
      • SIIG Cyber Serial Dual PCI (シリアル×2)
      • SIIG Cyber Serial PCI (シリアル×1)
      • VScom PCI-800 (8ポートシリアル、多分OEM)
      (類似のボードをサポートするのは単純であることが多いです。 そのボードが借りられたら、数分の作業で動作させられるようになるかもしれません。)
    • Cyclades Cyclom-Y シリアルボード (cy)
    • Universal Serial Bus コントローラー。USB Open ホストコントローラー (ohci) と USB Universal ホストコントローラー (uhci)を含む。
  • ディスクとCD-ROMコントローラー
    • MFM, ESDI, IDE, RLL の ハードディスク・コントローラー (pciide) (サポートされている ATA デバイス)、 (サポートされている ATAPI デバイス)。 バスマスターDMAは以下の PCI IDE チップセットでサポートされています:
      • Intel 82371FB (PIIX)
      • Intel 82371SB (PIIX3)
      • Intel 82371AB (PIIX4)
      • CMD Technology PCI0643
      • CMD Technology PCI0646
      • VIA Technologies VT82C586 (Apollo VP)
      • VIA Technologies VT82C586A
      • Contaq Microsystems (Cypress) CY82C693
      • Silicon Integrated Systems 5597/5598
      • Acer Labs M5229
  • サウンドハードウェア:
    • Ensoniq AudioPCI (eap)
    • ESS Solo-1 PCI AudioDrive (eso)
    • S3 SonicVibes (sv)
  • ビデオカード(VGAか、それ以上)
    • カーネルは基本的なコンソール機能をサポートします。
    • X は XFree86によってサポートされます。
  • 標準 PCI-PCI ブリッジ。PCI 拡張バックプレーンを含む。 (ppb)
