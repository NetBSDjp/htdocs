Macro System Development

MacroSystem Computer GmbH は、 Commodore Amiga マシン用のオーディオ、 グラフィック、ビデオボードと、ビデオ編集ソフトウェアで有名でした。 Commodore が破産した時、彼らは自分たちでマシンを作ることを決定し、 DraCo (いくつかのデバイスドライバーを変更した AmigaOS が動く68060 マシン) と Casablanca (ビデオ編集専用にいくつかの機能が省略されています)を作りました。

DraCo は低速の周辺機器(例えば、ネットワーク、増設のシリアルボード等) にZorro-2 バスを使用します。そして、 グラフィックとビデオボードには、バッファーされたローカルバス を使用します。

NetBSD/Amiga は DraCo マシン上でも動作するように拡張されました。 DraCo でブートする最初の公式リリースは NetBSD-1.2 でした。
