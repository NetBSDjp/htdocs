ADB

Apple Macintosh マシンで使用されている Apple Desktop Bus (ADB) に 関する情報へのリンクです。

  • 「Inside Macintosh: デバイス」の ADB Manager の章
    主に MacOS ADB Manager について記述されています。しかし、非常に低レベル で、ADB 上のデバイスとコミュニケートする方法についても書かれています。
  • ADB-The Untold Story
    Apple Technical Note は、いくつもの Apple Desktop Bus に関する難解かつ 未知の問題について説明しています。
  • Apple Desktop Bus Q&A
    ADB バスにかんする Apple の Q&Q のアーカイブ。おもに MacOS 固有の話です。
