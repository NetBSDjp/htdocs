バス

ここにリストされているのは、NetBSD ユーザーとハッカーが興味を持つで あろうバスです。もし、バスがリンクを持っていなければ、現在はそのバス に関する興味深い情報が無いということです。もし、役に立つリンクを見 つけたら、どうか 私達に教えてください!

  • ADB - Apple Desktop Bus
  • ATA - AT Attachment (ISA/IDE の拡張)
  • DEBI - Acorn RiscPC 等のマシンのメインインターフェースカードバス。
  • EISA - Extended Industry Standard Architecture
  • HP-IB - Hewlett-Packard Interface Bus
  • I2C - Philips I2C
  • IEEE1394 - Firewire、 i.Link
  • ISA - Industry Standard Architecture
  • Massbus - DEC、 VAX 700 シリーズ マシンで使用されています。
  • MCA - MicroChannel Architecture
  • NuBus - Apple の NuBus
  • PCI - Peripheral Component Interconnect
  • PDS - Apple の 'Processor-Direct Slot'
  • Q-bus - PDP、VAX、いくつかの DECstation で使用されています。
  • SCSI - Small Computer System Interface
  • Sbus - Sun Sbus
  • SDI - Standard Drive Interconnect
  • TurboChannel - DEC、VAXstation、DECstation、Alpha で使用されています。
  • Unibus - DEC、 VAX 700 シリーズ マシンで使用されています。
  • USB - Universal Serial Bus.
  • VAXBI - DEC、VAX 6000 と 8000 シリーズで使用されています。
  • VL-Bus - VESA Local Bus (ISA の拡張、当初は高速なビデオのために開発された)
  • VME - VMEbus
  • XMI - DEC、VAX 6000、7000、9000、10000 シリーズで使用されています。
  • Zorro - Commodore の Amiga bus。
