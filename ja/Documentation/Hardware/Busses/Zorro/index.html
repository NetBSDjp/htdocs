Zorro

Commodore の Amiga マシンとMacro System Development の DraCo で使用されている Zorro バスに関する 情報へのリンクを集めています。

Zorro バスには二つのタイプがあります。

  • Zorro 2: すべての Amiga マシンで使用されています。一部のバージョン (若干タイミングが速く、DMA が無い)は DraCo でも使用されています。

  • Zorro 3: A3000 と A4000 Amiga マシンで使用されています。

Jens Schroder は Zorro 2 と Zorro 3 の両方のバス に関する情報のページをもっています。
