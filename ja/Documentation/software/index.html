NetBSDのためのアプリケーションソフトウェア

商用ソフトウェア

NetBSDには所定のポートに他のオペレーティング・システムのためにコンパイルされたプログラムを走らせることができるエミュレーションシステムを持っています。 サポートされるエミュレーションはＣＰＵによって変わります。

さらにNetBSDで利用可能な 商用ソフトウェア のギャラリーがあります。

NetBSDパッケージシステム (pkgsrc)

NetBSDパッケージコレクションは NetBSDシステムに何百というサードパーティーソフトウェア パッケージをインストールするための最も簡単な方法です。 パッケージシステムは特定のパッケージが依存するライブラリまたはモジュールを自動的に インストールします。
  • カテゴリー毎に分類された利用可能なソフトウェア
  • アルファベット毎に分類された全てのパッケージ
  • NetBSDパッケージコレクション(pkgsrc)について.
  • パッケージシステムの成長.

他のサードパーティーソフトウェア

たいていの他のUNIXソフトウェアはダウンロードでき、他のUNIXシステムにコンパイル、インストールできます。 それは NetBSDパッケージシステムのソフトウェアのようにスムーズではありませんが、一般に比較的容易です。
