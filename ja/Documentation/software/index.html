商用ソフトウェア

NetBSDには所定のポートに他のオペレーティング・システムのためにコンパイル されたプログラムを走らせることができる エミュレーションシステムを持っています。 サポートされるエミュレーションは CPUによって変わります。

さらにNetBSDで利用可能な 商用ソフトウェア のギャラリーがあります。

NetBSD パッケージシステム (pkgsrc)

NetBSDパッケージコレクションは NetBSDシステムに何百というサードパーティー ソフトウェアパッケージをインストールするための最も簡単な方法です。 パッケージシステムは特定のパッケージが依存するライブラリーまたはモジュールを 自動的にインストールします。
  • カテゴリー毎に分類された利用可能なソフトウェア
  • アルファベット毎に分類された全てのパッケージ
  • NetBSDパッケージコレクション(pkgsrc)について。
  • パッケージシステムの成長。

他のサードパーティーソフトウェア

たいていの他のUNIXソフトウェアはダウンロードでき、他のUNIXシステムのように コンパイル、インストールできます。 それは NetBSDパッケージシステムのソフトウェアのようにスムーズではありませんが、 一般に比較的容易です。
