NetBSD 1.3.3 Release Announcement

Date: 23 Dec 1998 03:53:35 -0000
From: cgd@netbsd.org
To: netbsd-announce@netbsd.org
Subject: NetBSD 1.3.3 released

The NetBSD Project is pleased to announce the release of NetBSD 1.3.3.

This release fixes a substantial number of bugs found in the 1.3.x family of releases, including security issues. We urge users on the Internet to upgrade their systems.

It includes significant new or updated features, such as an updated FTP client and updated package tools, and updates our version of the X Window System from XFree86 release 3.3.2 to XFree86 3.3.3.

Finally, it also includes a somewhat improved installation procedure for multiple ports.

The release binaries and source can be found at:

ftp://ftp.netbsd.org/pub/NetBSD/NetBSD-1.3.3/

and on mirror sites across the world. For a list of mirror sites, see the file:

ftp://ftp.netbsd.org/pub/NetBSD/NetBSD-1.3.3/MIRRORS

For a partial list of changes to the main system sources since 1.3.2 (just 'src'; not including X or package source changes), see:

ftp://ftp.netbsd.org/pub/NetBSD/NetBSD-1.3.3/CHANGES-1.3.3

For a patch containing all of the changes to the main system sources since 1.3.2, see:

ftp://ftp.netbsd.org/pub/NetBSD/NetBSD-1.3.3/source/patches/diff-1.3.2-to-1.3.3.gz
