NetBSD 1.3.3 リリース声明

Date: 23 Dec 1998 03:53:35 -0000
From: cgd@NetBSD.org
To: netbsd-announce@NetBSD.org
Subject: NetBSD 1.3.3 released

NetBSD プロジェクトは、 NetBSD 1.3.3 のリリースを発表できることを喜ばしく思います。

このリリースでは、 1.3.x 系リリースに見つかった、セキュリティー上の問題を含む 多くのバグを修正しています。インターネット上のユーザーは、 システムをアップグレードするよう強くおすすめします。

更新された FTP クライアントやパッケージツール、 X Window System のバージョンが XFree86 release 3.3.2 から XFree86 3.3.3 に上がるなど、 重要な新機能や更新を含んでいます。

最後に、このリリースは、複数のポート向けにいくらか改良された インストール手順も含んでいます。

リリースバイナリーとソースは、以下の場所にあります:

ftp://ftp.NetBSD.org/pub/NetBSD/NetBSD-archive/NetBSD-1.3.3/

および世界中のミラーサイト。ミラーサイトの一覧は、 以下のファイルを見てください:

ftp://ftp.NetBSD.org/pub/NetBSD/NetBSD-archive/NetBSD-1.3.3/MIRRORS

1.3.2 からのメインシステムソースの変更点の部分的な一覧 ('src' のみ; X やパッケージソースの変更は含まない) は、以下のファイルを見てください:

ftp://ftp.NetBSD.org/pub/NetBSD/NetBSD-archive/NetBSD-1.3.3/CHANGES-1.3.3

1.3.2 からのメインシステムソースの変更をすべて含んだパッチは、 以下の場所にあります:

ftp://ftp.NetBSD.org/pub/NetBSD/NetBSD-archive/NetBSD-1.3.3/source/patches/diff-1.3.2-to-1.3.3.gz
