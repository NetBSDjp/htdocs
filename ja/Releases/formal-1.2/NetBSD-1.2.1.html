NetBSD 1.2.1 Release Announcement

To: netbsd-announce@NetBSD.ORG
Subject: Announcing release of NetBSD 1.2.1
Date: Tue, 20 May 1997 20:12:10 -0700
From: "J.T. Conklin" <jtc@NetBSD.ORG>

I am pleased to announce the release of NetBSD 1.2.1.

The current release is a maintenence release containing hundreds of fixes for bugs and security holes. This release is intended for people who run production systems on NetBSD 1.2.

In addition to full sources, binary releases are available for the amiga, atari, hp300, i386, mac68k, mvme68k, sparc, and x68k ports.

NetBSD 1.2 is available for download via ftp from ftp.netbsd.org in the directory pub/NetBSD/NetBSD-1.2.1. It will be available from the NetBSD mirror sites shortly.

       --jtc
