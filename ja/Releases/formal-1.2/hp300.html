NetBSD/hp300 1.2

NetBSD/hp300 1.2 は NetBSD/hp300 の 3 番目の公式リリースです。

メンテナンスリリースである NetBSD 1.2.1 が 1997 年 5 月 20 日に提供されました。 新機能はありませんが、数多くのバグやセキュリティーホールを修正しています。 このページでは、 1.2 と 1.2.1 の両方について述べます。

Jason Thorpe が、 1.2 リリース時点での NetBSD/hp300 のメンテナーです。

対応ハードウェア

NetBSD/hp300 1.2 は、ほとんどの HP 9000/300- および 400- シリーズのマシンで 動作します。テストされている最小の RAM は 4M です。 X を動作させるのであれば、より多くの RAM を推奨します。

フルインストール向けに推奨される HD パーティションサイズ:

パーティション推奨値最低限
root (/) 15M 10M
user (/usr)80M 45M
swap (2 or 3 * RAM)6M (miniroot)
(あとはすべてあなた次第です!)

注意: hp300 のインストール過程では、ディスクのスワップ領域に置かれた `miniroot' ファイルシステムを使います。この miniroot イメージが十分 収まる大きさのスワップパーティションが必要です。

NetBSD/hp300 1.2 では、以下のデバイスに対応します:

  • CPU:
    • 68020 ベース: 318、 319、 320、 330 および 350
    • 68030 ベース: 340、 345、 360、 370、 375 および 400
    • 68040 ベース: 380、 425 および 433
  • ディスク:
    • HP-IB/CS80: 7912、 7914、 7933、 7936、 7945、 7957、 7958、 7959、 2200 および 2203
    • SCSI-I、光磁気ディスクと CD-ROM を含む
  • テープドライブ:
    • 低密度 HP-IB/CS80 カートリッジ: 7914、 7946 および 9144
    • 高密度 HP-IB/CS80 カートリッジ: 9145
    • HP-IB/CS80 1/2": 7974A、 7978A/B、 7979A、 7980A および 7980XC.
    • SCSI: HP DAT および Exabyte 、また、 Archive Viper のような SCSI QIC ドライブ
  • RS232 インターフェース:
    • 98644 内蔵シングルポート (dca)
    • 98642 4-port (dcm)
    • 98638 8-port (dcm)
  • ネットワークインターフェース:
    • 98643 内蔵および追加の LAN カード
  • ディスプレイ:
    • 98544、 98545 および 98547 カラーおよびモノクロ Topcat
    • 98548、 98549 および 98550 カラーおよびモノクロ Catseye
    • 98700 および 98710 Gatorbox
    • 98720 および 98721 Renaissance
    • 98730 および 98731 DaVinci
    • A1096A モノクロ Hyperion
  • 入力デバイス:
    • あらゆる HIL デバイスをサポートした汎用インターフェース: キーボード、 2 および 3 ボタンマウス、および ID モジュール
  • その他いろいろ:
    • バッテリーバックアップされたリアルタイムクロック
    • 98624 内蔵 HP-IB インターフェース。
    • 98625A および 98625B "fast" HP-IB インターフェース。
    • 98658A 内蔵 SCSI インターフェース
    • RS232 および HP-IB 上のプリンターおよびプロッター
    • SCSI オートチェンジャー
ここに挙がっていないデバイスでも、これらと互換性があるかもしれませんし、 あるいは単に私たちが書き忘れているだけかもしれません。 うまく動かなければ、あなたがドライバーを書いてもよいでしょう !

注意

  • Domain/OS 用に設定された 400 シリーズのマシン (400、 425、 433) には完全には対応していません。特に、キーボードが動作せず、 ROM は `HP-UX mode' にしかなりません。
  • SCSI-II ドライブは動作するのですが、 ドライブによってはジャンパーの変更が必要になります。 詳細はディスクのドキュメンテーションを読んでください。
  • `HIL to quad' コンバーターに繋がれたシリアルマウス も、動作することがわかっています。

対応ブートオプション

NetBSD/hp300 1.2 は、 SCSI および HP-IB ディスクから (SYS_PBOOT) と、ローカルエリアネットワーク経由 (SYS_NBOOT) のブートに対応します。

もし NetBSD/hp300 に興味があれば、 より新しいバージョンを使うべきです。
