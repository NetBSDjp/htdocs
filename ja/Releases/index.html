Formal Releases

These are are intended to provide a stable, supported platform for use in production systems. The latest formal release is NetBSD 1.3.2.
Formal releases are available:
  • On CD-ROM.
  • Download via the Internet.

NetBSD-current

This is a daily snapshot of the NetBSD development source tree. It is for people who want to be on the bleeding edge of NetBSD development. It is only available off the Internet. People working with local changes will probably want to import the sup changes into a CVS tree.
