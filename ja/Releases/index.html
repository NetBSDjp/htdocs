NetBSDの入手法

リリース番号付け情報については、 NetBSDリリース用語集と図 をご覧ください。

公式リリース

公式リリースは、安定した、製品システムとして使用できる サポートされたプラットフォームを提供することをめざしています。 最新の公式リリースは NetBSD 1.5.2です。 NetBSD-release ブランチは最近の公式リリースの(リリース後に行なわれた変更を含む) 最新の状態を手に入れる簡単な方法として考えられます。

  • FTP でのNetBSD のダウンロード
  • 公式リリースを含んだCD-ROM。
  • ダウンロード可能なISO イメージ。
  • 最新と 一つ前の公式リリースの 詳細情報

NetBSD-current

バイナリースナップショットは、個々の ports 用に用意されています。 通常、これらは公式リリースより多くの機能を提供しますが、 同程度のテストはされていないので、 可能ならば製品システムに使用するのは避けるべきです。
バージョンは常に数字の後ろに文字がついた形式になります (例:1.5G)。

  • -current を含んだCD-ROM (一覧表を入念に調べて下さい)。
  • ミラーサイトとその他の ダウンロードの方法。
  • ftp.netbsd.org の スナップショット置き場
  • NetBSD-current の追跡、 たとえば AnonCVS ミラーから AnonCVS を利用、 またはsup を利用。
  • NetBSD-current の詳細情報。

web によるソースコードの閲覧

cvsweb インタフェースを用いると、 手元にチェックアウトしたコピーを持つことなく リポジトリー中の任意のソースへのアクセスや、任意のバージョン間の diff 生成が容易に行なえます。 ミラーサイト の一覧を用意してあります。
