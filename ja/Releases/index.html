How to get NetBSD

For more information on release numbering, please see the NetBSD release glossary and graphs

Formal Releases

These are intended to provide a stable, supported platform for use in production systems. The latest formal release is NetBSD 1.6.1. The NetBSD-release branch can be considered an easy way to get the most up to date fixes for the most recent release.

  • Download NetBSD via FTP.
  • Order CD-ROMs containing formal releases.
  • ISO-images available for download.
  • More information on current and previous formal releases.

NetBSD-current

The Release Engineering server runs nightly autobuilds for every port of -current. Generally these provide more features than formal releases, but have not had the same degree of testing, and should be avoided on production systems where possible. The release engineering server also has a web server with autobuild statistics. A list of mirror sites is also available.
The version is always in the form of a number followed by a letter (eg: 1.6G).

  • Order CD-ROMs containing -current (check listing carefully).
  • Mirror sites, and other methods of downloading.
  • Snapshot area on ftp.NetBSD.org.
  • The ftp area of the Release engineering server and the release engineering webserver. A list of mirror sites is also available.
  • Tracking NetBSD-current, e.g. via AnonCVS using one of our AnonCVS mirrors, or sup.
  • More information on NetBSD-current.

Browsing source code on the web

The cvsweb interface provides access to easily retrieve files or diffs of any version in the source repository without having to keep a checked-out copy around locally. A list of mirror sites is also available.
