How to get NetBSD

For more information on release numbering, please see the NetBSD release glossary and graph

Formal Releases

These are are intended to provide a stable, supported platform for use in production systems. The latest formal release is NetBSD 1.4.
Formal releases are available:
  • On CD-ROM.
  • Download via the Internet.

NetBSD-current snapshots

Binary snapshots may be available for individual ports. Generally these provide more features than the formal releases, but have not had the same degree of testing, and should be avoided on production systems where possible.
NetBSD-current snapshots are available:
  • On some CD-ROMs (check listing carefully).
  • Download via the Internet.
It is probably worth reading through the tracking -current documentation.
