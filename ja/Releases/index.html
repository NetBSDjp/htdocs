How to get NetBSD

For more information on release numbering, please see the NetBSD release glossary and graph

Formal Releases

These are intended to provide a stable, supported platform for use in production systems. The latest formal release is NetBSD 1.4.2. The NetBSD-release branch can be considered an easy way to get the most up to date fixes for the most recent release.

  • CD-ROMs containing formal releases.
  • Downloading NetBSD 1.4.2 from ftp.netbsd.org.
  • Mirror sites.
  • More information on formal releases.

NetBSD-current

Binary snapshots may be available for individual ports. Generally these provide more features than formal releases, but have not had the same degree of testing, and should be avoided on production systems where possible.
The version is always in the form of a number followed by a letter (eg: 1.4G).

  • CD-ROMs containing -current (check listing carefully).
  • Mirror sites, and other methods of downloading.
  • Snapshot area on ftp.netbsd.org
  • Tracking NetBSD-current.
  • More information on NetBSD-current.

Browsing source code on the web

The cvsweb interface provides access to easily retrieve files or diffs of any version in the source repository without having to keep a checked-out copy around locally using the AnonCVS service.
