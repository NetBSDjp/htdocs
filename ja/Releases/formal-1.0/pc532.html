NetBSD/pc532 1.0

NetBSD/pc532 1.0 は、 NetBSD/pc532 の最初の公式リリースです。

Phil Nelson が、 1.0 リリース時点での NetBSD/pc532 のメンテナーです。

対応ハードウェア

最小構成の NetBSD/pc532 1.0 システムは 4M の RAM と約 40M のディスクスペースを必要とします。フルインストール (ソースを含む) には、少なくとも 8M の RAM と 200M のディスク容量が推奨されます。

NetBSD/pc532 1.0 では下記のデバイスがサポートされます:

  • 8 serial lines done by 4 scn2681 chips
  • NCR DP8490 SCSI チップ (SCSI のみ)
    • ほとんどの SCSI ディスクが動作 (ハードディスクとフロッピーのいずれも)
    • いくつかの SCSI テープが動作
    • いくつかの SCSI CD-ROM が動作

もし NetBSD/pc532 に興味があれば、 より新しいバージョンを使うべきです。
