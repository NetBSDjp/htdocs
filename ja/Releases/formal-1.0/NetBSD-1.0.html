NetBSD 1.0 リリース声明

From: cgd@agate.berkeley.edu (Chris G. Demetriou)
Newsgroups: comp.os.386bsd.announce
Subject: Announcing the release of NetBSD 1.0
Date: 8 Nov 1994 11:17:59 -0800

自由に使用・再配布可能で多くのプラットフォームで動作する BSD システム、 NetBSD オペレーティングシステムリリース 1.0 の発表をお知らせできることを 喜ばしく思います。 NetBSD 1.0 は、 4.4BSD Lite と Berkeley Networking Release 2 のソースをもとにしています。 ほとんどのアーキテクチャー上の共有ライブラリーのサポートと、拡張された Berkeley Fast File System (64 ビットのファイルサイズとクラスタリングに対応) 、堅強なネットワークコード、他のオペレーティングシステムとの 高度なバイナリー互換機能を含んでいます。 前回のリリースからの重要な変更点のリストは、配布に含まれる "CHANGES" ファイルにあります。このリストは、ここに載せるには長すぎますので。

NetBSD オペレーティングシステムのリリース 1.0 は、 完全なソースコード一式とともに配布されており、 この記事の末尾に掲げるサイトから入手可能です。現時点では、 下記のアーキテクチャー用のバイナリーリリースがあります:

  • Amiga
  • i386 ファミリー PC
  • いくつかの m68k ベースの Macintosh
  • PC532
  • SPARCstation (sun4c) クラスのマシン

さらに、まだ用意できていませんが、 HP9000/300 系ワークステーション用の バイナリーリリースも予定されています。このほか、 NetBSD 1.0 のソースツリーには DECstations、 Sun 3 モデルの一部、 VAX の一部への対応が含まれていますが、 これらのポートは、完全なバイナリーリリースをするところにまでは 達していません。

今後数日のうちに、上記の各ポート別のインストールノートとリリースノートが、 各ポート別のメーリングリストやニューズグループに投稿される予定です。 これらのノートを読んでもあなたの NetBSD についての疑問が晴れない場合は、 数多くある NetBSD メーリングリストからどれかを購読して、 そこで質問すべきかもしれません。 どのようなメーリングリストがあるかと講読方法については、 メッセージボディーが空のメールを majordomo@NetBSD.ORG まで送ってください。

Enjoy!

Chris Demetriou
The NetBSD Project
