NetBSD 1.3 for sun3x

対応ハードウェア

NetBSD/sun3x 1.3 は、 Sun Microsystems 製 3x ファミリーのワークステーション (3/80、 3/460、 3/470 および 3/480 モデル) をサポートします。 3/80 デスクトップワークステーションのオンボードデバイスは、パラレルポート以外は、 すべてサポートしています。デスクトップワークステーション (3/460/470/480) は、 VME スロットによって拡張可能ですが、現在のところ 'si' SCSI カードと Intel 'ie' イーサネットアダプターを含む VME デバイスがサポートされています。

必要なハードウェア

NetBSD/sun3x を使うには、少なくとも以下のものが必要です。
  • 4MB のメモリー (X を使う場合は 8MB)
  • 50MB のディスクスペース (X を使う場合は 150MB)

対応ソフトウェア

NetBSD/sun3x は完全に機能するオペレーティングシステムであり、標準的な UNIX ユーティリティーが附属しています。さらに、他の m68k アーキテクチャーすべて (atari、 amiga、 hp300、 mac68k) とバイナリー互換であり、 また、 SunOS 4 エミュレーション機能も持っています。

パッケージ

NetBSD 1.3 では、バイナリーパッケージと FreeBSD から導入された「 ported (移植された) 」ソフトウェアコレクションが、新たにサポートされました。 パッケージサブシステムは FTP で入手できます。

入手方法

NetBSD/sun3x 1.3.2 は、数多くある NetBSD ftp ミラーサイトから FTP で入手できます。どうぞ INSTALL ドキュメントの説明をお読みください。
