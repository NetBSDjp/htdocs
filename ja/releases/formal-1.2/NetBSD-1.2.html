NetBSD 1.2 Release Announcement

To: netbsd-announce@NetBSD.org
Subject: Announcing the release of NetBSD 1.2.
Date: Fri, 04 Oct 1996 00:05:27 -0700
From: "J.T. Conklin" <jtc@NetBSD.org>

I am pleased to announce the long awaited release of NetBSD 1.2.

The current release has built upon the foundation laid by NetBSD 1.1 with new ports, new and improved kernel subsystems, updated development tools and libraries, assorted userland enhancements, and many bug fixes. The results of these improvements is a stable operating system fit for production use and ready for the next phase of development.

NetBSD 1.2 is available for download via ftp from ftp.NetBSD.org and its mirrors in the directory pub/NetBSD/NetBSD-1.2.

Many thanks go to the NetBSD developers for making this release possible.

Even more thanks go to the many NetBSD users who have supported NetBSD with their enthusiasm, evangelism, and contributions.

       --jtc
