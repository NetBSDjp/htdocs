NetBSD/x68k 1.2

NetBSD/x68k 1.2 は、 NetBSD/x68k の最初の公式リリースです。

沖勝が、 1.2 リリース時点での NetBSD/x68k のメンテナーです。

対応ハードウェア

NetBSD/x68k 1.2 は、 *完全な* 68030 プロセッサー (68EC030 は*ダメ*) と 数値演算コプロセッサーを持つ SHARP X680x0 シリーズの パーソナルワークステーションで動作します。 ページメモリー管理ユニット (PMMU) が必要です。 このほか、いくつかのアクセラレーターに対応しています:

  • Xellent30 シリーズ (with 68030)
  • 040turbo (68040)
  • JUPITER-X (68040)
最小の構成では 4M の RAM と約 65M のディスクスペースを必要とします。 完全なシステムのインストールにはより多くのディスクスペースが必要ですし、 X の動作やコンパイルにはより多くの RAM を推奨します。

NetBSD/x68k 1.2 では下記のデバイスがサポートされます:

  • Sharp 純正 MB89352 SCSI ホストアダプター
  • 内蔵フロッピードライブ
  • 内蔵フレームバッファー (テキストおよびグラフィックスクリーンいずれも)
  • 内蔵 Z8530 シリアル
  • 内蔵パラレル (セントロニクスプリンターインターフェース)
  • 標準キーボード
  • マウス / トラックボール
  • NS16550 UART
  • Neptune-X DS8390 イーサネット

よく尋ねられるが、現在サポートされていないハードウェア:

  • Mach-2 SCSI ホストアダプター
  • 68060 プロセッサーのサポート

将来はこれらのデバイスの多くをサポートしたいと考えています。

もし NetBSD/x68k に興味があれば、 より新しいバージョンを使うべきです。
