The NetBSD Foundation への寄付

寄付募集運動

皆さんから the NetBSD Foundation へ寄付していただくことで、 NetBSD プロジェクトではコードベースの大きな改良をすることができます。 私たちは、NetBSD 6.0 に向けた作業を始めるにあたり、今年末までに 60,000 米ドルを目標とする、 6.0 への寄付募集運動を発表します。私たちは、下記をはじめ、システムのさまざまな領域において、 資金援助をともなう開発を続けていきたいと考えています。

  • ネットワークスタックの並列性および性能改善。
  • 近代的なファイルシステムの開発、および、 既存ファイルシステムの改善。
  • 組み込み環境で使われる機能、たとえば、 高解像度のタイマーや execute in place (XIP) 対応など。
  • 自動的な試験、および、品質保証。

私たちは、以上の目標を達成するために、NetBSD コミュニティーや NetBSD を使っている組織に、協力をお願いしています。今回の寄付募集運動は、 NetBSD プロジェクトへ寄与し、 私たちが NetBSD を改良できるよう助けていただくための、よい機会なのです。

寄付の方法についての詳細は、 以下の説明をお読みください。皆さんの NetBSD プロジェクトへのご支援に感謝します。よろしくお願いします。

なぜ寄付が重要なのか

The NetBSD Foundation は、オープンソースソフトウェアを制作している 非商用のボランティア組織であり、米国内国歳入法 (Internal Revenue Code) 501(c)(3) の非営利組織です。 つまり、第三者からの寛大な寄付に頼るほかには、定期的な収入源などないということです。

The NetBSD Foundation への財政支援元は多岐にわたります。 NetBSD に関心を持つ 組織や個人が、the NetBSD Foundation の重要なサービスを提供するため、 また、開発者が新しいハードウェアプラットフォームや周辺機器をサポートするために、 資金、ハードウェア、ラックスペースを寄付してくださっています。

あらゆる種類の寄付について理解することが重要です。サーバーやラックスペースや コロケーション設備なしには、定期的な構築や、さまざまな web や FTP サービスを開発者やユーザーに提供することはできなかったでしょう。 開発用ハードウェアの寄付なしには、こんにち提供しているような広範な プラットフォームや 周辺機器 をサポートすることはできなかったでしょう。イベントで宣伝活動をすることで、 私たちの成果を外部に伝えることができ、あらたな開発者やユーザーを得ることができます。 これらはいずれも、the NetBSD Foundation の健全な運営のために重要なことです。

ですから、幸いなことに多くの企業が NetBSD を製品に組み込んでいますが、 これにかかわらず、 NetBSD でサーバーを運用している企業、デスクトップで NetBSD を使うホームユーザー、あるいは NetBSD がクールだと (でなくても何でも) お思いな方も、どうぞ the NetBSD Foundation への寄付をご検討ください。

もちろん、個人や組織が、 The NetBSD Foundation に貢献する方法は寄付以外にもあります。

The NetBSD Foundation に寄付をする

個人の寄付

個人の方が the NetBSD foundation に寄付してくださった場合、 お名前、web サイトと電子メールアドレスを NetBSD の web サイトに掲載させていただきます。

企業の資金提供

The NetBSD Foundation では、企業からの資金提供を大変ありがたく思っており、 NetBSD に寄付していただいた場合、 金額に応じて、以下の情報を掲出しています。

  • シルバースポンサーシップ ($100 以上 $1,000 以下)
    • 会社名 (web サイトへもリンクします)
    • 連絡先電子メールアドレス
  • ゴールドスポンサーシップ ($1,001 以上 $5,000 以下)
    • 会社名 (web サイトへもリンクします)
    • 連絡先電子メールアドレス
    • 会社のロゴ (150×350 ピクセル以下)
  • プラチナスポンサーシップ ($5,001 以上)
    • 会社名 (web サイトへもリンクします)
    • 連絡先電子メールアドレス
    • 会社のロゴ (350×500 ピクセル以下)
    • $10,001 以上の寄付をしていただいた場合は、 これに加えて、ホームページ http://www.NetBSD.org/ にロゴを掲出させていただきます。 複数の企業からそのような寄付をしていただいた場合、 24 時間ごとに切替えて掲出します。

寄付の方法

もっとも簡単な寄付方法は、Google Checkout や PayPal を使った送金です。

Google Checkout を使って送金するには、以下のフォームに金額を入力してください。

$
 

Google のアカウントを作成する必要があります (支払するときにアカウントを作成することができます)。 Google Checkout を使うと、あなたのクレジットカードから the NetBSD Foundation の口座に直接送金されます。

PayPal を使って送金するには、以下のボタンからリンクを辿ってください。

PayPal を使って NetBSD へ送金するために PayPal 口座を持っている必要はありません。 PayPal のサイトでクレジットカードを使って、 NetBSD の PayPal 口座へ直接送金することができます。 PayPal 口座をお持ちの方は、その口座から送金することもできます。 NetBSD の PayPal アドレスは <paypal@NetBSD.org> です。

これらの送金方法を使いたくない場合は、小切手の送付や銀行振込によることもできます。 送付先や振込先口座の詳細について、 <finance-exec@NetBSD.org> まで (訳註: 英語で) ご連絡ください。

The NetBSD Foundation への寄付はすべて、 米国内では完全に免税されます。

寄付に関する方針

The NetBSD Foundation には、寄付していただいた方への謝辞を web ページ (下記) に掲載する上での方針があります。 私たちは、寄付を受け取ると、IRS 向けの確認の節と、 web ページへの掲載方法 (資金提供がどの程度かによります) に関する簡単な質問事項を含む、 感謝状をお送りします。

ゴールドおよびプラチナスポンサーとなっていただく場合、 以下に収まる大きさ (X×Y) の JPEG 形式のロゴをご提供ください。

  • ゴールド: 150×350 ピクセル以内のロゴ
  • プラチナ: 350×500 ピクセル以内のロゴ
  • プラチナ ($10,000 を超える場合のみ): さらに、インデックスページ掲載用として、150×150 ピクセル以下のロゴ。

寄付の過程についてなにか質問があったり、 あなたの名前を貢献者の一覧に載せたい、 あるいは載せたくない場合には、 <finance-exec@NetBSD.org> で、 (訳註: 英語で) 私たちにお知らせください。
