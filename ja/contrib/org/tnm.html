The NetBSD Mission

The NetBSD Mission は、展示会および web 上における、 スタッフ、文献、宣伝アイテム、 ソフトウェア配布、さまざまな出費への資金提供のための組織です。

以下のようなイベントが、この Mission にサポートされました:

  • Usenix Annual Technical Conference 2001
  • Comdex Spring 2001
  • LinuxWorld Expo
  • Comdex Fall 2000
  • Linux Expo Toronto 2000
  • Atlanta Linux Showcase 2000
  • O'Reilly Open Source Software Convention 2000
  • Comdex Canada 2000
  • Usenix Annual Technical Conference 2000
  • Comdex Spring 2000
  • Linux Expo Montreal 2000
  • Linux Expo Paris 2000
  • The Bazaar
  • Comdex Fall 1999
  • Atlanta Linux Showcase 1999

 この Mission はおもに Charles Hannum によるコンサルトの収益に頼っていますが、 外部からの寄付を歓迎します。
