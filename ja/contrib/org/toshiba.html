東芝

東芝は、互換テストのためのハードウェア貸与や、また、 東芝製コンピューター上での NetBSD の互換性の詳細を含む machine/OS compatibility database を整備することで、支援してくれています。

以下のものを含む開発作業が、東芝からのハードウェアによって支援されました:

  • usb モデムコードのデバッグ
  • 3COM HomeConnect Ethernet USB Adapter (3C460) の認識および 独自ドライバーの利用可能化
  • cardbus ドライバーフレームワークの作業
  • 大容量メモリーサポート (4GB RAM) の正常動作の確認
  • S3 Savage/MX を装着したマシンの S3 Savage ベースのカードに対する xsrc でのサポート
  • cdparanoia を Toshiba ATAPI DVD ドライブで使うための調整
  • cd-rw 操作のドキュメンテーション
  • Cirrus Logic CS4281 ドライバー開発中のテスト
  • I2O および AMI RAID ドライバーのテストおよび開発
どうか注意: この情報は NetBSD プロジェクトによって「現状のまま」で提供され、 明示または暗示を問わず、いかなる保証もしません。
