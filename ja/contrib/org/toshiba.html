Toshiba

Toshiba has supported with loans of hardware for compatibility testing and also maintains a machine/OS compatibility database that includes details about compatibility for NetBSD on Toshiba computers.

Development work that was aided by hardware from Toshiba include:

  • debugging the usb modem code
  • recognition of the 3COM HomeConnect Ethernet USB Adapter (3C460) and enabling the proper driver to use it
  • work on the cardbus driver framework
  • ensuring large-memory support (4GB RAM) works properly
  • providing a S3 Savage/MX equipped machine to enable the support for S3 Savage based cards in xsrc
  • adjusting cdparanoia to run on Toshiba ATAPI DVD drives
  • documentation of cd-rw operation
  • testing during development of Cirrus Logic CS4281 driver
  • testing and development of I2O and AMI RAID drivers
