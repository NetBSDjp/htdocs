March 2002
2002-03-25: NetBSD/mac68k switches to ELF
NetBSD/mac68k is now using ELF as its native object file format in -curr ent. With the COMPAT_AOUT_M68K kernel option, all your old a.out binaries should still work. Note that a.out versions of ifconfig(8) and route(8) will not work due to some structure alignment problems.

A NetBSD/mac68k ELF snapshot is available in the mac68k snapshot area of the ftp server. Also available are instructions for upgrading to ELF from this snapshot or from source.

2000-02-14: New console driver integrated

September 1999
1999-09-25: New Mkfs released
A new version of Mkfs (the NetBSD installation utility for making filesystems under the Mac OS) was posted at: ftp://ftp.NetBSD.org/pub/NetBSD/arch/mac68k/installation/. The major change in this version allows a clean installation of NetBSD over an old Linux install. Sources for this version are also available.
June 1999
1999-06-27: Alternate interrupt scheme for Quadras
At least some of the 68040-based macs have an alternate interrupt mapping that can be enabled--this is now enabled, much improving the clock skew for these machines.

1999-06-26: Quadra 900/950 serial console
The Quadra 900 and Quadra 950 are usable from the serial console as of today.
May 1999
1999-05-30: FPE updated
The 68881/68882 floating point emulator fixes from PR 7220 (submitted by Ken Nakata) have been applied. The FPE should be fully functional for the 020 and 030 systems, but is expected to still have some problems on the LC040. Contributions welcome.

1999-05-12: NetBSD-1.4 released
The NetBSD 1.4 release supports sixteen architectures, and was distributed free by Usenix to all 1999 ATC attendees.
