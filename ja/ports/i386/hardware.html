NetBSD/i386 対応ハードウェア

NetBSD/i386 が動作するには最低 4M の RAM とおよそ 40M のディスクの 空き容量が必要です。 完全なインストール(ソースと X11 を含みます)には 少なくとも 8M の RAM と 200M のディスクの空き容量が 推奨されます。

全ての Intel 純正 あるいは Cyrix, AMD, NexGen などの互換 CPU の内、 i386 以上の CPU なら動作するはずです。

NetBSD/i386がサポートしているデバイス (ドライバーの名前は括弧内に示されています) は 次の通りです。

  • 機種非依存のISA デバイスドライバー
  • 機種非依存のPCI デバイスドライバー
  • 機種非依存のEISA デバイスドライバー
  • 機種非依存のPCMCIA デバイスドライバー
  • 機種非依存のCardbus デバイスドライバー
  • 機種非依存のATA デバイスドライバー
  • 機種非依存のSCSI と ATAPI デバイスドライバー
  • 機種非依存のUSB デバイスドライバー
  • 機種非依存のMII デバイスドライバー (IEEE 802.3 Media Independent Interface, FastEthernet コントローラーの 物理的メディアインターフェース等に用いられています。)
  • 機種非依存のMCA デバイスドライバー
  • 標準の PC フロッピーコントローラー(fdc)とドライブ(fd)
  • アダプテック社の 284x VLbus SCSI ホストアダプター (ahc)
  • NCR社の 53C80 port-mapped および memory-mapped SCSI ホストアダプター (nca)
  • PC スピーカー (pcppi および sysbeep)
  • ロジテック社のバスマウス (lms)
  • マイクロソフト社の InPort マウス (mms)
  • PS/2 マウス (pms)
  • ゲームポート上のジョイスティック (joy)
  • MDA, CGA, VGA, SVGA 及び HGC ディスプレーアダプター (pccons または wscons). 注意: NetBSD/i386 でサポートされている全てのディスプレーアダプターで X が動作するわけではありません。 詳しくは XFree86 FAQを ご覧下さい。
