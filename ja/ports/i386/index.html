NetBSD/i386

NetBSD/i386 について

NetBSD/i386はi386ファミリープロセッサーを搭載した IBM PCとPC互換機用のNetBSDポートです。 数値演算コプロセッサーの有無に関わらず、ISA(ATバス)、EISA、 PCI、VLバス そして MCAシステムで動作します。

NetBSD/i386はNetBSDのオリジナルポートで、初のリリースは 1993年のNetBSD 0.8です。

Andrew Doran が NetBSD/i386のメンテナーです。

NetBSD/i386 ニュース

2007-01-07:   Xbox サポートを -current に統合
Microsoft Xbox のサポートが、Andrew Gillham と Jared D. McNeill によって -current に統合されました。 ブート可能な ISO イメージ、ルートファイルシステムのサンプル、および説明は、 NetBSD ftp サーバーのここに用意されています。
2006-04-09:   i386 用バイナリーパッケージが利用可能
Manuel Bouyer が、 NetBSD/i386 2.1 および 3.0 用の新しいバイナリーパッケージが ftp.NetBSD.org に用意されたことを発表しました。 このパッケージは、最新の枝である pkgsrc-2006Q1 から構築されたものです。
2003-10-23:   pmap(9) の改良
Niels Provos が -current の i386 pmap(9) に変更を加え、 リンクリストのかわりに splay tree (自己調整二分木) を使うようにしました。 この結果、プロセス数増大時に fork(2) がよりよくスケールするようになりました。
