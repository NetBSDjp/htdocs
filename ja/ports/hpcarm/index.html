NetBSD/hpcarm

NetBSD/hpcarm について

NetBSD/hpcarm は Intel StrongARM および XScale ベースの Windows CE および Windows Mobile PDA マシンで動作する NetBSD オペレーティングシステムです。

“hpcarm” というポート名は、 NetBSD/hpcarm が動作する ハードウェア: マイクロソフト社の H/PC (ハンドヘルド PC) 仕様に準拠している StrongARM および XScale ベースのシステム、に由来します。 H/PC マシンはキーボードとタッチスクリーン、そして 8MB か それ以上の RAM を備えています。このポートは H/PC のほかに、 H/PC Pro と PsPC (パームサイズ PC) にも対応しています。 (多くの PsPC マシンが開発者達の実際の開発に使われています)

このハードウェアの大きさからして、 モバイルコンピューティング用アプリケーションを開発するときに NetBSD/hpcarm は良いプラットフォームとなるでしょう。 もちろん、 NetBSD/hpcarm は IPv6 や IPsec といった NetBSD の 標準的な機能の全てに対応しています。

NetBSD/hpcarm ニュース

2010-08-23:   NetBSD/hpcarm W-ZERO3 用インストール済みディスクイメージ
2GB の SD カードに収まる NetBSD/hpcarm W-ZERO3 用インストール済みディスクイメージが、http://ftp.NetBSD.org/pub/NetBSD/misc/tsutsui/wzero3/ にあります。 面倒なインストール作業をしなくても、 UNIX 環境で gzip(1) と dd(1) を使って (または、新版が出たばかりの Windows 用 Rawrite32.exe ツールを使って)、このイメージを 2GB の SD カードに書き込むだけで、 X サーバーつきの NetBSD/hpcarm を試しに使ってみることができます。 このディスクイメージの NetBSD/hpcarm 5.99.38 用に、 pkgsrc-2010Q2 をもとにしたコンパイル済みバイナリーパッケージも用意してあります。 詳細は、README ファイル (または README.ja) をご覧ください。
2010-08-08:   W-ZERO3 マシンの LCD 速度改良
フレームバッファーメモリーに対してライトスルーキャッシュを有効にすることで、 W-ZERO3 マシンの LCD コンソールの速度が改良され、最大で 3 倍速くなりました。
2010-07-31:   標準で Xorg サーバーを使うよう変更
NetBSD/hpcarm ポートは、標準状態で Xorg サーバーを使うようになりました。 スナップショットバイナリーは 日々の自動構築サーバーにあります。 詳細は、発表のメールをご覧ください。
2010-07-17:   disklabel の MAXPARTITIONS 増加
disklabel(5) でのパーティション数の上限が 8 から 16 に増加し、i386 や他の hpc ポートと同じになりました。
2010-07-05:   WS003SH および WS004SH の内蔵無線 LAN に対応
WS003SH および WS004SH で使われている Intersil PrismGT 802.11b/g WLAN に対応しました。

WS011SH 用キーパッドのドライバーが追加されました。

WS011SH および WS020SH 用の Xorg キーボード構成ファイルが追加されました。
2010-05-23:   WS020SH 対応の更新
SD の脱着検知と、WS020SH 用キーマップが追加されました。

WS011SH 用タッチパッドのドライバーが追加されました。

WS003SH および WS007SH 用の Xorg キーボード構成ファイルが追加され、 Xorg サーバーが CCW (横画面) で動作するようになりました。
2010-05-13:   WS003SH 用タッチパッド
WS003SH 用タッチパッドのドライバーが追加されました。

WS004SH, WS007SH, WS011SH で X サーバーが動作します。詳細は、蛯原純による netbsd-advocacy の記事をご覧ください。

対応システム

NetBSD/hpcarm は多くの種類の H/PC, H/PC Pro や PsPC マシンに対応しています。 あなたのシステムが対応しているかどうかは対応システム表 をご覧下さい。

対応プロセッサー

NetBSD/hpcarm は StrongARM および PXA2x0 XScale プロセッサーに対応しています。 あなたのプロセッサーが対応しているかどうかは プロセッサー比較表をご覧下さい。
