NetBSD/cobalt

NetBSD/cobalt について

NetBSD/cobalt は、 Cobalt Networks の MIPS ベースの Microservers (現在では Sun Server Appliances となっています) 向けのNetBSD ポートです。

NetBSD/cobalt ニュース記事のアーカイブ

対応機種

Gateway Micro ServersとCobalt Networksから出ているMIPSベースの Qube、およびRaQシリーズの全てがサポートされます。

Phoenix Firewall Appliance Microserver と、いくつかのバージョンの Seagate NasRaQ もサポートされます。

インストールには、ビルトインのシリアルポートが必要です。

RaQ 3/4 ノート

i386ベースのRaQ 3、RaQ 3i、およびRaQ 4モデルはcobalt ポートでは サポートされません。
