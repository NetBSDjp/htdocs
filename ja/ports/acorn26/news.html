NetBSD/acorn26 news archive

March 2004
2004-03-01: NetBSD 1.6.2 released
NetBSD 1.6.2 released with support for 40 architectures. More information is available in the 1.6.2 release announcement.
April 2003
2003-04-21: NetBSD 1.6.1 released
NetBSD 1.6.1 released with support for 40 architectures. More information is available in the 1.6.1 release announcement.
September 2002
2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.
December 2001
2001-12-08: New -current snapshot available
A new -current snapshot based on 2001-12-07 sources is now available in the NetBSD/arm26 snapshot area of the ftp server.
September 2001
2001-09-12 Minimal Econet support
NetBSD-current now includes minimal support for Econet interfaces. At the moment, there are no protocols implemented on top of Econet, so its usefulness is rather limited. This will change.
July 2001
2001-07-28 New bootloader
Ben Harris has committed a new bootloader, with support for gzipped kernels, to NetBSD/arm26. See Ben's announcement in the port-arm26 mail archive for more details.
May 2001
2001-05-31 boot with local root
Ben Harris (bjh21@NetBSD.org) has gotten support for booting with root on a local disk.

2001-05-28: DMA in HCCS SCSI driver
Ben Harris (bjh21@NetBSD.org) has committed support for the pseudo-DMA facilities of the HCCS SCSI podule. This gets speed up to 300 KB/s.
December 2000
2000-12-30: eh(4) driver
A new driver for i-cubed EtherLan 100-, 200-, and 500-series Ethernet cards is available.
October 2000
2000-10-13: New snapshot available
A new snapshot based on 2000-10-05 -current sources is now available in the NetBSD/arm26 snapshot area of the ftp server.
July 2000
2000-07-23: Ether3 driver
The ea driver for SEEQ 8005-based Ethernet cards (usually known as Ether3s) is now minimally usable.

2000-07-20: FOURMEG kernel
The 20000714 snapshot now includes an additional kernel configuration, FOURMEG, which I hope will be small enough to boot on systems with 4Mb of RAM.

2000-07-16: 20000714 binary snapshot
The first proper binary snapshot of NetBSD/arm26 has been uploaded. See the announcement on port-arm26 for details.

2000-07-13: arm26 cross compiler in pkgsrc
The NetBSD package system now contains a set of cross-build tools for NetBSD/arm26 in the cross/armv2-netbsd package.
May 2000
2000-05-09: arm26 port committed
The arm26 port has now been merged into the main NetBSD source tree. Well, mostly, anyway.
