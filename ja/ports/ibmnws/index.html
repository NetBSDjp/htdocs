NetBSD/ibmnws

About NetBSD/ibmnws

NetBSD/ibmnws は PowerPC ベースの IBM Network Station Series 1000 コンピューターへの NetBSD ポートです。

対応システムモデル

Model IBM Number CPU Memory NIC NetBSD Release
IBM Network Station 1000 8362-A22 PowerPC 603 32M Token-Ring -
IBM Network Station 1000 8362-A23 PowerPC 603 64M Token-Ring -
IBM Network Station 1000 8362-A52 PowerPC 603 32M Ethernet 10/100 -
IBM Network Station 1000 8362-A53 PowerPC 603 64M Ethernet 10/100 -
