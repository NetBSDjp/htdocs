NetBSD/sgimips

NetBSD/sgimips について

NetBSD/sgimips は、 MIPS プロセッサーベースの Silicon Graphics 製コンピューター向けの NetBSD ポートです。

NetBSD は、 Silicon Graphics O2 で動作する世界初のオープンソース オペレーティングシステムです。

NetBSD/sgimips ニュース

2009-02-12:   IP6 および IP10 サポートがソースツリーに
-current で、 Personal IRIS 4D20 (IP6) および 4D/25 (IP10) に新たに対応しました。
2007-04-14:   wscons が O2 に対応
IP32 (つまり O2) 上の SGI-CRM グラフィックスコントローラーと MACE にもとづくキーボード・マウスに対する wscons の対応が、ソースツリーに入りました。
2006-12-30:   複数のバグ修正と機能追加
この休暇中に、sgimips ポートに対して、E++ GIO イーサネットアダプター、 Set Engineering GIO ファストイーサネットカード、Indigo Light (LG1/LG2) フレームバッファー、IP12 上のタイムカウンターへの対応をはじめとする、 複数のバグ修正と機能追加がおこなわれました。 バグ修正は、おもに IP12 ポートに集中しています。ほとんどの変更は -current でおこなわれており、NetBSD 4.0 リリースには含まれそうにありません。
2004-07-11:   O2 のオンボード MACE MAC-110 イーサネット向けの機能するドライバー
IP32 (O2) のオンボード MACE MAC-110 イーサネットのサポートが commit されました。
2004-07-08:   Indigo 向けの wscons サポートがソースツリーに
IP20/IP12 (つまり Indigo マシン) の GR2 グラフィックコントローラーおよび z8530 ベースのキーボード/マウスに対する wscons サポートがソースツリーに入りました。
2004-04-11:   IP12 サポートがソースツリーに
IP12 (4D/30, 4D/35, Indigo R3k) サポートがソースツリーに入りました。詳しくは、 Antti Kantee の記事 をご覧ください。
2003-12-31:   Indigo (IP20) サポートがソースツリーに
Indigo (IP20) サポートがソースツリーに入りました。 NFS ルートおよびローカルルートのどちらでも マルチユーザーにまで上がります。詳しくは、 Christopher Sekiya の記事 をご覧ください。
2003-12-15:   newport およびコンソール接続のコミット
Ilpo Ruotsalainen が、 SGI NG1 (``newport'') グラフィックコントローラーの ドライバーをコミットしました。詳しくは、彼のemail をご覧ください。
2003-10-15:   O2 サポート復活
O2 (IP32) サポートが、 Christopher SEKIYA による修正の寄与により、 ふたたび使い物になるようになりました。さらに、 R10000 ベースの O2 のサポートも追加されました。

NetBSD/sgimips ニュース記事のアーカイブ

対応システムモデル

  • IP6 マシン
    • 4D/20
  • IP10 マシン
    • 4D/25
  • IP12 マシン
    • Indigo (R3000)
    • 4D/30 (理論上/未確認)
    • 4D/35
  • IP20 マシン
    • Indigo (R4x00)
  • IP22 マシン
    • Indigo2 (R4x00)
    • Challenge M
  • IP24 マシン
    • Indy (R4x00, R5000)
    • Challenge S (R4x00, R5000)
  • IP32 マシン
    • O2 (R5000, RM5200, R10000, R12000)

対応周辺機器

  • オンボード Z8530 シリアルインターフェース (zsc)
  • オンボード SEEQ 80c03 イーサネットインターフェース (sq)
  • オンボード WD33C93 SCSI インターフェース (wdsc)
  • オンボード HAL2 オーディオインターフェース (haltwo)
  • O2 のオンボード AIC7880 wide SCSI インターフェース (ahc)
  • O2 のオンボード NS16550 ベースシリアルインターフェース (com)
  • O2 のオンボード MACE MAC-110 イーサネット (mec)
  • O2 オンボードフレームバッファー (crmfb) (-current)
  • O2 のオンボード PS/2 キーボード/マウス (macekbc, pckbd, pms) (-current)
  • O2 のオンボード Moosehead オーディオインターフェース (mavb) (-current)
  • Indy のオンボードフレームバッファー (newport)
  • Indy のオンボード PC 形式キーボード/マウス (pckbc, pckbd, pms)
  • Indigo のオンボードフレームバッファー (grtwo, light)
  • Indigo のオンボード Z8530 ベースのキーボード/マウス (zsc, zskbd, zsms)
  • E++ GIO イーサネットアダプター (sq) (-current)
  • GIO32 SCSI アダプター (wdsc) (-current)
  • Phobos GIO G100/G130/G160 ファストイーサネット (tlp) (NetBSD 4.0/-current)
  • Set Engineering GIO 100baseTX ファストイーサネット (tl) (-current)
  • ほとんどの MI PCI デバイス
  • MI SCSI デバイス

未対応のもの

機種 / デバイス / 機能:

  • R10000 Power Indigo2 (IP28) (作業進行中)
  • Octane (IP30) (mips64 toolchain が必要)
  • R4600 CPU を持つマシン上の L2 キャッシュ
  • その他いろいろ
