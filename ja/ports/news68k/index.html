NetBSD/news68k

NetBSD/news68k について

NetBSD/news68k は、 SONY の m68k ベースの “NET WORK STATION” シリーズ向けの NetBSD ポートです。これらのマシンの追加情報を用意しました。

NetBSD/news68k ポートは、NetBSD/newsmips と NetBSD/m68k ポートをもとにしています。

NetBSD/news68k ニュース

2006-05-09:   m68k 用のバイナリーパッケージが利用可能

Greg Oster が最新の枝 pkgsrc-2006Q1 から構築した約 1000 個のパッケージが ftp://ftp.NetBSD.org/pub/NetBSD/packages/packages-2006Q1/NetBSD-3.0/m68k で入手できます。 このパッケージは m68k ベースの全ポートで使うことができます。

詳細は、Greg Oster のお知らせをご覧ください。

NetBSD/news68k ニュース記事のアーカイブ

対応機種

現在のところ、 NetBSD/news68k は mc68030 ベースの シングルプロセッサーの機種のみに対応しています。

  • NWS-1750
  • PWS-1560
  • NWS-1450
  • NWS-1460
  • NWS-1530
  • NWS-1580
  • PWS-1550
  • NWS-1250 (長らくテストされていません; 支援者求む)

対応周辺機器

  • オンボード LANCE イーサネットインターフェース (le(4))
  • オンボード SONY CXD1180 SCSI インターフェース (si(4))
  • ほとんどの SCSI デバイス
  • キーボード (kb)、マウス (ms)
  • オンボード Z8530 シリアルインターフェース (zsc(4)) (動作するフレームバッファーがまだないため、 シリアルコンソールにのみ対応)

未対応のもの

機種 / デバイス / 機能

  • mc68020 ベースのシステム (NWS-700/800)
  • IOP ベースのシステム (NWS-800/1800)
  • NWS-1200 の CXD1185 SCSI インターフェース
  • 内蔵フロッピーディスク
  • フレームバッファー (作業中ですが、 ハードウェア情報が必要です)
  • COMPAT_NEWSOS
