NetBSD/evbarm news archive

November 2005
2005-11-02: NetBSD 2.1 released
NetBSD 2.1, the first maintenance release of the netbsd-2 release branch, has been released with binary distributions for 48 architectures. More information is available in the 2.1 release announcement.
December 2004
2004-12-09: NetBSD 2.0 released
NetBSD 2.0 released with support for 48 architectures. More information is available in the 2.0 release announcement.
March 2004
2004-03-01: NetBSD 1.6.2 released
NetBSD 1.6.2 released with support for 40 architectures. More information is available in the 1.6.2 release announcement.
May 2003
2003-05-23: Support for NOVATEC NTNP425B board committed
Ichiro FUKUHARA has written and committed support for the NOVATEC NTNP425B("ZAO425"), an evaluation and development platform for the Intel IXP425 XScale Core NetworkProcessor.
April 2003
2003-04-21: NetBSD 1.6.1 released
NetBSD 1.6.1 released with support for 40 architectures. More information is available in the 1.6.1 release announcement.
January 2003
2003-01-24: Support for ADI Engineering BRH board committed
Jason Thorpe has written and committed support for the ADI Engineering BRH ("Big Red Head"), an evaluation and development platform for the Intel i80200 XScale processor. The port was contributed by Wasabi Systems, Inc.

November 2002
2002-11-20: Support for Samsung SMDK2800 board committed
Hiroyuki Bessho has written and committed support for the Samsung SMDK2800, Samsung's reference platform for the S3C2800 ARM920T based processor. The port was contributed by Fujitsu Component Ltd., and Genetec Corp.
September 2002
2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.
July 2002
2002-07-15: Support for Intel IXM1200 board committed
Ichiro FUKUHARA and Naoto Shimazaki have written and committed support for the Intel IXM1200, Intel's reference platform for the IXP1200 Network Processor.
March 2002
2002-03-27: Support for Intel IQ80321 board committed.
Jason Thorpe has written and committed support for the Intel IQ80321, Intel's reference platform for the IOP321 I/O Processor. The port was contributed by Wasabi Systems, Inc.
February 2002
2002-02-27: Support for Team ASA, Inc. Npwr committed
Jason Thorpe and Allen Briggs have written and committed support for the Team ASA, Inc. Npwr, an IOP310-based embedded system product. The port was contributed by Wasabi Systems, Inc.
November 2001
2001-11-07: Support for Intel IQ80310 board committed
Jason Thorpe and Allen Briggs have written and committed support for the Intel IQ80310, Intel's reference platform for the IOP310 I/O Processor chipset. The port was contributed by Wasabi Systems, Inc.
October 2001
2001-10-27: NetBSD/evbarm port committed
The NetBSD/evbarm port is committed to the source tree. The port initially supports the ARM Ltd. Integrator board. The port was done by Richard Earnshaw, and contributed by ARM, Ltd.
