NetBSD/cats

NetBSD/cats について

NetBSD/cats は、 Simtec Electronics CATS マザーボードおよび Intel EBSA ボードへの NetBSD ポートです。

NetBSD 1.6 のリリースまでは、このポートは NetBSD/arm32 ポートの一部でした。 NetBSD 1.6 のリリースに伴い、 cats は、 arm の中核機能を使って独立したポートとして保守されるようになります。

NetBSD/cats ニュース

2003-10-15:   arm ポートの Xscale 最適化
Steve Woodford が、 NetBSD/arm ポートに さまざまな Xscale への細かな最適化をコミットしたことを発表しました。 詳細は、彼が port-arm メーリングリストに投稿した記事をご覧ください。
2003-10-09:   cats の ABLE ファームウェアの実験的サポート
Chris Gilbert が、 ABLE ファームウェアの実験的なサポートを cats に追加しました。 さらなる情報は、 port-cats メーリングリストに彼が投稿したメール をご覧ください。
2002-09-13:   cats の新しいファームウェア
Simtec が cats 用の新しいファームウェアをリリースしました。 インストールするようおすすめします。ある人のところでは発生した X に関する問題は、これにより修正されます。 この新しいファームウェアは、Simtec の web サイト、 具体的にはcats サポートページにあります。 cats ユーザーは 1.3 にアップグレードするよう強くおすすめします。 このファームウェアは大きなカーネルを扱うことができるので、いくつかのデバイスを -current の INSTALL カーネルでまた有効にできるからです。
2002-08-19:   cats 用 X サポート
NetBSD/cats のツリー内に xfree86 4.2.0 サポートが入りました。1.6 RC1 用のものが ftp.NetBSD.org の pub/NetBSD/arch/cats/xsets 以下に用意されています。X を試したければ最新の xsrc に更新する必要があります。サーバーが /dev/pci0 インターフェースを使うため、 1.6BETA または 1.6 -current が必要です。 -current 以外のユーザーは /etc/mk.conf に USE_XF86_4=YES を書き足す必要があります。netbsd-1-6 枝への pullup 要求をしていますが、 1.6 には入れてもらえないかもしれません。
2001-07-29:   Cats のメーリングリストができました。
NetBSD/cats 専用のメーリングリスト port-cats ができました。このリストは、 cats と EBSA 、およびこれらに関連する議論を扱います。 arm 全般に関することは、これまでどおり port-arm リストを使うべきです。 参加するには、このページ脇のリンクを辿ってください。
2001-06-10:   Cats が専用の arch ディレクトリーへ移動
NetBSD/cats は、専用の arch ディレクトリー sys/arch/cats に入るようになりました。 カーネルはこれをもとに構築できるはずです。 GENERIC は現在サポートされている USB デバイスすべてを含むようになり、膨れていることに注意してください。 cats 固有のバグはすべて、 send-pr で新たに作られた port-cats カテゴリーに 報告すべきです。

対応コンピューターシステム

このポートでは、以下のマシンとボードを対象としています。

  • Simtec の cats
  • Intel EBSA 評価ボード

このほか、GXemul は NetBSD/cats ポートを動かすこともできるマシンエミュレーターです。
