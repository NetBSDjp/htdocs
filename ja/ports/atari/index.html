NetBSD/atari

About NetBSD/atari

NetBSD/atari は、 Atari シリーズのパーソナルコンピューターへの NetBSD ポートです。 NetBSD/atari の開発活動は、開発者の余暇に応じたスピードで 進んでいます。現在、 NetBSD/atari は TT030、 Falcon および Hades で動作します。 Milan 向けの実験的なサポートもあります。

NetBSD/atari News

2010-04-01:   EtherNEC に対応
EtherNEC イーサネットへの対応が、 NetBSD-current に commit されました。 EtherNEC は、Atari の ROM カートリッジスロットに接続する NE2000 ベースのイーサネットデバイスで、 古い Atari マシンでネットワークを使う方法のなかで最も入手性がよいものです。
2010-03-13:   SMC_TT VME-ISA ブリッジを介した SMC Elite Ultra ISA イーサネットカードに対応
SMC_TT VME-ISA ブリッジを介した SMC Elite Ultra イーサネット への対応が、 NetBSD-current に commit されました。 SMC_TT は、TT030 の 16 ビット VME スロットと SMC Elite Ultra ISA イーサネットカード用の、 ユーザーが設計した“手作りの” VME-ISA ブリッジ回路です。 SMC_TT とこのドライバーを装備した TT030 は、 オープンソースカンファレンス 2010 Kansai@Kobe の NetBSD ブースでも実演されました。 TT030 の GIF アニメによる実演をご覧ください。
2009-01-23:   最近の NetBSD/atari の変更および修正点

NetBSD/atari を再び動くようにするため、最近、いくつかの修正がおこなわれました。 最近の変更点の要約は以下のとおりです。

  • ncr5380 SCSI ドライバーが、再び機能するようになりました (プローブ時にフリーズしない)
  • Ramdisk ドライバーが修正されました (インストールに必須)
  • インストーラーとブートローダーの問題が修正されました
  • Falcon カーネルが 68060 CPU (CT60/63) に対応しました
  • カーネル構成ファイルが更新されました
  • 割り込みハンドラーの問題が修正されました
  • Amiga pmap の変更が Atari に取り込まれました (yamt-km のマージに必要)
  • Falcon IDE ドライバーが割り込みを取りこぼす問題が修正されました

対応ハードウェア

最低限、システムには 68030 CPU、 4MB の RAM (うち 2MB は ST-RAM で可)と、 SCSI または IDE ディスクが必要です。配布されている BOOT および BOOTX カーネルはいずれも FPU エミュレーションを含んでいるので、 FPU は実際には必須ではありません。現状でのエミュレーションは MC68882 命令セットを完全にカバーしてはいませんが、非常に使えるシステムであると 認めていただけるでしょう。

  • TT-HIGH を含む、 ST および TT ビデオモード
  • Falcon ビデオ (Direct Color - 15/16 bit depth は除く)
  • Hades et4000/w32-pci ビデオアダプター
  • Hades et6000-pci および et6100-pci ビデオアダプター
  • 内蔵 5380 SCSI アダプター
  • ほとんどの SCSI ディスク、 CD-ROM、テープおよび ZIP ドライブ
  • リアルタイムクロック
  • SCC シリアルポート (serial2/modem2)
  • 720Kb/1.44Mb フロッピードライブ
  • パラレルプリンター
  • Falcon および Hades 上の IDE インターフェース (ATAPI を含む)
  • ひとつ目の 68901 UART (modem1) 上のシリアルインターフェース
  • Falcon (CT60/63) および Hades における 68060 をサポート
  • Falcon FX メモリー拡張
  • atari マウス
  • 3 ボタンマウス (作り方参照)
  • ROM カートリッジスロット上の EtherNEC イーサネット (-current)

  • サポートされている VME バスデバイス (TT030/Hades)

    • VME BVME410 イーサネット
    • Circad Leonardo 24-bit VME グラフィックアダプター
    • Crazy Dots VME et4000 グラフィックアダプター
    • VME Riebl (and possibly PAM) イーサネット
    • SMC_TT VME-ISA ブリッジを介した SMC Elite Ultra ISA イーサネット(-current)

  • サポートされている Hades PCI デバイス

    • Adaptec 2940U SCSI NOT (下記参照)
    • ESS Technology Inc. Solo-1 サウンドカード
    • 3Com 3c59x ネットワークカード

  • サポートされている Hades ISA デバイス

    • teles 16.3 カードでの I4BSD サポート
    • NE2000 互換カード

  • サポートされている Milan PCI デバイス

    • Intel EtherExpress PRO 10+/100B

Note

  • Hades PCI バスは非常にクリティカルです。ほとんどのカードは認識されません。 これは電気的な問題が原因のようです。
  • 現在の Adaptec ドライバーは、 残念ながら、もはや動作しません。
