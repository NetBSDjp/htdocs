NetBSD 1.4.2 Security Patches

Below is the list of patches applicable to the NetBSD 1.4.2 release:

  • A description and a patch for the IP Options processing bug can be found in NetBSD Security Advisory SA2000-002
  • A description and a patch for the semcontrol(2) vulnerability can be found in NetBSD Security Advisory SA2000-004
  • A description and a patch for a local "cpu-hog" denial of service can be found in NetBSD Security Advisory SA2000-005
  • A description and a patch for the /etc/ftpchroot parsing bug can be found in NetBSD Security Advisory SA2000-006
  • A description and a patch for a security vulnerability in the DHCP client can be found in NetBSD Security Advisory SA2000-008
  • A description and a patch for a security vulnerability in the FTP server can be found in NetBSD Security Advisory SA2000-009
