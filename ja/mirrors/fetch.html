NetBSD WWW Mirror fetch list

This document links to the handful of files on www.NetBSD.org that are generated by www.NetBSD.org. These files get generated daily from content in the doc module of the NetBSD CVS repository.

In the example crontab in the WWW Mirror portion of the NetBSD Mirror FAQ, you'll see 

wget -B http://www.NetBSD.org/ -nv -xnH -N -Fi mirrors/fetch.html

This means that daily, through wget, these www.NetBSD.org created files will be updated in your copy of the web site contents.

Documentation on setting up a NetBSD WWW mirror is available in the NetBSD Mirror FAQ.

Note

Any documents referred to by hyperlinks on this page have to be listed in a .cvsignore file in their directory.
