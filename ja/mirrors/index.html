NetBSD Mirror Sites

NetBSD Mirrors are available from the following sites as of December 30, 2001.

If you wish to become a distribution site for NetBSD, please refer to the NetBSD Mirror FAQ for more information.

Services:

  • FTP
  • ISO images
  • Anonymous CVS
  • SUP
  • CVSup
  • rsync
  • AFS
  • NFS
  • WWW (Website mirrors - not for downloading NetBSD)
  • CVSweb (browse the source code on the web)

Note: These links won't take you to any particular release of NetBSD, but rather to that site's ``root'' NetBSD distribution area.

Please use the mirror site closest to you.
