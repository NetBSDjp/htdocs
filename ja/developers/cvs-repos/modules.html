CVS - Top-level modules

The top-level directories in the NetBSD CVS repository related to the NetBSD sources (i.e. the sources shipped to users) and their contents are:

src
The main NetBSD system sources, that is, the contents of the system source tree excluding those pieces in other top-level directories in the CVS repository (listed here).
pkgsrc
The NetBSD packages collection.
xsrc
The NetBSD X Window System source tree.

Implications of this repository layout

  • Extra care must be taken when importing files. Imports which involve code going into multiple top-level directories must be broken into a separate import per top-level directory. If you encounter other problems, please update this page!
