CVS - トップレベルモジュール

NetBSDソース (つまりユーザーに渡されるソース) に関するNetBSD CVS リポジトリーがあるトップレベルのディレクトリーと内容は以下の通り:

src
NetBSD の主要システムソース。つまり、(ここで挙げられている) CVS リポジトリーの他のディレクトリーにあるものを除く、 システムソースツリーのコンテンツ。
pkgsrc
NetBSDパッケージコレクション。
xsrc
NetBSD X Window System ソースツリー。

このリポジトリー配置の密接な関係

  • ファイルをインポートするときには更なる注意を払わなければいけません。 複数のトップレベルディレクトリーへ置かれるコードを含むインポートは、 トップレベルディレクトリーごとにインポートを分けなければいけません。 もし他の問題に遭遇したら、このページを更新してください。
