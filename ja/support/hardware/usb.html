NetBSDでサポートされている USB ハードウェア

機種に依存しない USB ドライバー

下記のデバイス(括弧内はドライバー名)が機種に依存しない USB ドライバーによってサポートされています。 リストにないデバイスは、各プラットフォームのページをチェックしてみてください。 そのデバイスのドライバーが、まだ機種独立になっていないのかもしれません。

別の可能性として、そのデバイスはリストにあるデバイスと互換かもしれませんし、 そのデバイスを見落していて、リストに載せ忘れているのかもしれません。 不明な点がある時は、 -current のカーネルを起動するか、 <netbsd-users@NetBSD.org> にメールしてください。

訳註

netbsd-users は英語のメーリングリストです。日本語での問い合わせは、 ここを参照してください。

このページはNetBSD-currentの現状を反映していることに注意してください。 特定のリリースでサポートされているハードウェアを調べるには、 そのリリースのインストレーション・ノートを参照してください。

サポートされている USB デバイス

  • USB ハブ (uhub(4))
  • USB キーボード (ukbd(4))
  • USB マウス (ums(4))
  • USB タッチパネル (uep(4))
  • USB 汎用ヒューマン・インターフェース・デバイス (何でもあり) (uhid(4))
  • USB プリンター (ulpt(4))
  • USB モデム (ukyopon(4), umodem(4))
  • USB 大容量記憶デバイス、 例えば、 USB Zip ドライブ (umass(4))
  • USB オーディオ (uaudio(4))
  • USB MIDI (umidi(4))
  • USB スキャナー (uscanner(4), usscanner(4))
  • USB シリアルポート (ubsa(4), uftdi(4), umct(4), uplcom(4), uvscom(4))
  • USB イーサネットアダプター (aue(4), axe(4), cdce(4), cue(4), kue(4), udav(4), url(4))
  • USB 802.11 アダプター (atu(4), ral(4) および rum(4))
  • USB-USB ケーブル (upl(4))
  • USB 汎用ドライバー (ugen(4))
  • Handspring Visor (uvisor(4))
  • Y@p Phone (uyap(4))
  • Diamond Rio 500, 600 (urio(4))
  • USB IrDA ブリッジ (uirda(4) ustir(4))
  • USB ラジオ (udsbr(4))
  • iPAQ の USB インターフェース (uipaq(4))
  • USB Bluetooth コントローラー (ubt(4))
