NetBSD のサポート

NetBSD のサポートのなかで最も重要なソースは、NetBSD コミュニティーそのものです。コミュニティーの節では、 問題に出くわしたときに助けを求める先について、 一覧を掲載しています。

また、商用サポートが必要な場合には、 雇用コンサルタントも多数あります。

対応ハードウェア

個々のハードウェアの部品に対して、 NetBSD が対応しているかどうか知りたい場合は、対応ハードウェアデバイスの一覧を確認してください。

問題報告

バグがありましたか? どうぞお知らせください。

報告を実際に送信する前に、 PR (問題報告) データベースを検索して、 その問題がすでに報告されていないか、ご確認ください。

  • オンラインでの、バグ報告の送信
  • PR データベースの照会

検索

お探しの情報が見つかりませんでしたか? NetBSD の検索では、 お探しの情報を見つけるために役立つ方法を、いろいろ提案しています。

セキュリティー

NetBSD のセキュリティーについての詳細と、セキュリティー勧告は、 セキュリティーと NetBSD の web ページにあります。

情報とニュース

  • pkgsrc.se は簡単に辿ることのできる pkgsrc のフロントエンドです。pkgsrc と pkgsrc-wip の変更や更新を記録しています。 このサイトでは、"Package tracker" (パッケージごとの変更点を知らせる電子メール用)、 pkgsrc の統計情報や、RSS フィードも提供しています。 これは「pkgsrc のすべて」ともいえるサービスです。

  • NetBSD News Beat は NetBSD に関する各種ニュースフィード、ブログなどの集積です。 あなたの RSS フィードをここに追加したい場合は、 <hubertf@NetBSD.org> へメールを送ります。
