X Window System

  • X Window System あるいは X11 は、 NetBSD で使われるグラフィカルディスプレイシステムです。NetBSD では、プラットフォームによって、 X.Org または XFree86 のいずれかの実装が標準となっていますが、いずれの場合も、 これらのかわりに pkgsrc から X.Org を使うこともできます。 X11 に関するこの HOWTOは、 個別の設定の問題に役立つでしょう。
