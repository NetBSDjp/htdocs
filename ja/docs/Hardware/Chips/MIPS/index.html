ハードウェアドキュメンテーション - チップ - MIPS

MIPS

MIPS プロセッサーに関する情報へのリンクです。 Digital DECStation、 ARC、 Cobalt、そしてSGI マシンで使用されています。

  • MIPS の出版物

    インストラクションセット、プロセッサーユーザーマニュアル、アセンブリ 言語のプログラマーズマニュアルがあります。

  • MIPS ABI 仕様書

    MIPS プロセッサー ABI 適合ガイド。version 1.0 から最新のものまでがあります。

  • MIPS プログラマーズハンドブック

    アセンブリと C 言語の観点から見たMIPS アーキテクチャーについての 完全な説明が含まれています。組み込みアプリケーションに関する問題 については特に重点がおかれています。まえがきといくつかの例だけを オンラインで見ることができます。R3000 についても説明されているようです。

  • MIPS ソフトウェアシミュレーター (SPIM)

    X Window システムのインターフェースを持つインストラクションレベルのシミュレーターです。

  • PMC-Sierra 製品ページ

    RM7000 と RM52xx (R5000 互換) の情報が含まれています。

  • R4x00 インターフェースデザインガイド [PostScript ファイル]

    MIPS 64 ビット R4x00 プロセッサーのシステムインターフェースに関する 設計指向のドキュメンテーション。

  • SGI Techlibs Library

    ここには、 いくつかの MIPS マニュアルが含まれています。
