NetBSD/bebox クロスコンパイル環境

クロスコンパイル環境のインストール

  1. pkgsrc.tar.gz を入手して /usr/pkgsrc に展開します。
  2. make してインストールします。
    % cd /usr/pkgsrc/cross/powerpc-netbsd
    # make install clean clean-depends
  3. ppc-make スクリプトを入手します。

カーネルの作り方

  1. current のソースを入手・展開します。
  2. config して make します。
    % cd src/sys/arch/bebox/conf
    % config GENERIC
    % cd ../compile/GENERIC
    % ppc-make depends; ppc-make

ブートフロッピーの作り方

NetBSD/bebox 用ブートフロッピーディスクの作り方の説明 を見てください。

ユーザーランドバイナリーの作り方

  1. DESTDIR ターゲットを決め、 ppc-make の DESTDIR=.. を書き換えます。
  2. make して DESTDIR ディレクトリーにインストールします。
    # cd src
    # ppc-make beforeinstall
    # ppc-make includes
    # (cd lib/csu; ppc-make; ppc-make install)
    # (cd lib; ppc-make; ppc-make install)
    # ppc-make; ppc-make install
    エラーが出る場合は、その部分のターゲット (bin/ksh、 usr.bin/tn3270 などがありえます) をコメントアウトします。
