NetBSD/bebox cross compile environment

Install cross compile environment

  1. Get pkgsrc.tar.gz and extract on /usr/pkgsrc.
  2. Make and install.
    % cd /usr/pkgsrc/cross/powerpc-netbsd
    # make install clean clean-depends
  3. Get ppc-make script.

How to make kernel

  1. Get/extract current source.
  2. Config and Make.
    % cd src/sys/arch/bebox/conf
    % config GENERIC
    % cd ../compile/GENERIC
    % ppc-make depends; ppc-make

How to make boot floppy

See Description of making bootable floppy disk for NetBSD/bebox.

How to make userland binary

  1. Decide target DESTDIR, and substitute DESTDIR=.. in ppc-make.
  2. Make and install into DESTDIR.
    # cd src
    # ppc-make beforeinstall
    # ppc-make includes
    # (cd lib/csu; ppc-make; ppc-make install)
    # (cd lib; ppc-make; ppc-make install)
    # ppc-make; ppc-make install
    if error occured, comment out those target (maybe bin/ksh, usr.bin/tn3270).
