NetBSD/sh5

About NetBSD/sh5 

NetBSD/sh5 is the port of NetBSD to the SuperH SH-5 family of 32/64-bit CPUs.

Currently status is "Experimental port".

While it may be experimental, the port is stable in multi-user mode on the Cayman evaluation board, in both 32- and 64-bit modes.

Supported Systems 

  • SuperH Cayman SH-5 evaluation board (evbsh5)


Supported Peripherals 

  • SH-5 on-chip devices (superhyway)
    • Peripheral Bridge (pbridge)
      • Interrupt Controller (intc)
      • Clock, Power, Watchdog/Reset Controller (cprc)
      • Timer Unit (tmu)
      • Serial Port (scif)
      • Real-time Clock (rtc)
    • Flash/External Memory Interface (femi)
    • PCI Controller (sh5pci)
      • Devices connected via PCI controller.

  • Cayman-specific devices
    • System FPGA (sysfpga)
    • SMC Ethernet Controller (sm)
    • SuperIO Controller (superio)
      • ISA bus (isa)
        • Serial Ports (com)
        • Printer Port (lpt)
