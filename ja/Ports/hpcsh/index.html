NetBSD/hpcsh

About NetBSD/hpcsh

NetBSD/hpcsh brings the NetBSD operating system to Hitachi Super-H family based Windows CE PDA machines. Currently, the sh3 processor is supported.

The port name "hpcsh" comes from the name for some of the hardware that NetBSD/hpcsh runs on: SH based systems in Microsoft's H/PC (Handheld PC) form factor. A H/PC machine contains a keyboard and a touch screen and generally has 8MB or more of RAM. The port supports the HPC form factor, as well as the H/PC Pro and PsPC (Palmsized PC) form factors. (Many PsPC machines are used by the developers for active development.)

Because of the hardware platforms it supports, NetBSD/hpcsh is an ideal platform to use for mobile computing applications. And, of course, NetBSD/hpcsh supports all of the standard features you expect from NetBSD, such as IPv6 and IPsec.
