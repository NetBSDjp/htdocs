NetBSD/acorn32

NetBSD/acorn32 について 

NetBSD/acorn32 は 1.6 でその最初の公式リリースが予定されており、 これは ARM および StrongARM ベースの ARM6 以上の上で動作する Acorn RiscPC 互換マシンへのポートです。 Reinoud Zandijk によって保守されています。 Acorn への NetBSD 1.5.x のサポートは NetBSD/arm32 でおこなわれています。

このポートは、 1994 年に RiscBSD として最初にあらわれ、のちに NetBSD/arm32 へと変化しました。 「 arm32 」なるものは存在しません。最初の ARM プロセッサー (ARM2 と ARM3) は、 Acorn が設計し、 26 ビットの制限と貧弱な MMU を持っていました。これらのプロセッサーは NetBSD/arm26 でサポートされています。 のちに Acorn は Apple と VLSI とともに ARM を興しました。 ARM の CPU (6, 7, 8, 9 および StrongARM) は完全に 32 ビットで、 これらが NetBSD/arm32 から派生したポート群でサポートされています。

その後、 Digital は、より速い CPU である StrongARM を生産するため、 ARM アーキテクチャーのライセンスを 受けました。この技術は Intel に移転され、現在ではこの CPU は Intel StrongARM として知られており、その後継が Intel Xscale (StrongARM-II) です。


NetBSD/acorn32 ニュース 

2001-11-20: NetBSD/acorn32 作成
NetBSD/acorn32 が NetBSD/arm32 から分離され、 Reinoud Zandijk によって ARM チームの一部として保守されるようになりました。

2001-04-02: RiscPC/A7000/NC wscons 向けのハードウェアスクロール
Reinoud Zandijk は RiscPC および互換機向けのハードウェアスクロールの コードをコミットしました。この新しいコードは、異なる画面解像度と色数で テストされ、いずれもうまく動作しています。そして、オリジナルの vidcconsole コードよりも速く動くようにもなっています。より詳しくは、 port-arm32 メールアーカイブの Reinoud のアナウンスを参照してください。

NetBSD/acorn32 ニュース記事のアーカイブ


サポートされている ARM および StrongARM CPU 

ここに掲げるのは、 NetBSD/arm32 でのサポートがある CPU その他の部品です。 ここではビデオコントローラー、 core logic 、 PCI ブリッジ、 浮動小数点演算ユニットなどをリストしています。

  • ARM600, ARM610
  • ARM700, ARM710
  • ARM810
  • ARM7500FE
  • ARM VIDC
  • ARM FPA10, FPA11
  • ARM IOMD
  • Intel SA-110


サポートされているコンピューターシステム 

ここに掲げるのは、完全にサポートされている、さまざまなコンピューター メーカーによるシステムです。

  • Acorn Risc PC 600, Risc PC 700
  • Acorn StrongARM Risc PC, J233
  • Acorn A7000, A7000+
  • Acorn network computers (partial as yet)

下記のシステムは NetBSD 1.5 では NetBSD/arm32 でもサポートされていましたが、 NetBSD 1.6 では別ポートに分離される予定です。すでに NetBSD-current では分離されています。

  • Digital DNARD ("Shark") は、 NetBSD/dnard でサポートされるようになりました。
  • Chalice Technology CATS は、 NetBSD/cats でサポートされるようになりました。

初期の ARM ベースの Acorn マシンは NetBSD/arm26 ポートでサポートしています。 ARM ベースの Windows CE PDA は NetBSD/hpcarm でサポートしています。 StrongARM ベースの NetWinder マシンは、 NetBSD/netwinder でサポートしています。


人々 

コアプロセッサーのサポートは Mark Brinicombe (その大部分) と Neil Carson (libkern 、 pmap 、浮動小数点エミュレーター、 ライブラリーのいくつか) によって書かれました。

Acorn プラットフォームのサポートは、 Mark Brinicombe (その大部分) によって、 Scott Stevens (さまざまなデバイスドライバー) 、Melvin Tang Richardson (VIDC コンソール) 、 Mike Pumford (EtherH ドライバー) および Andrew McMurry (Acorn FileCore ファイルシステム) の協力を受けて、 書かれました。
