NetBSD/x68k news archive

April 2003
2003-04-21: NetBSD 1.6.1 released
NetBSD 1.6.1 released with support for 40 architectures. More information is available in the 1.6.1 release announcement.
September 2002
2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.
July 2002
2002-07-22: NetBSD 1.5.3 released
NetBSD 1.5.3 released with support for 20 architectures. More information is available in the 1.5.3 release announcement.
September 2001
2001-09-13: NetBSD 1.5.2 released
NetBSD 1.5.2 released with support for 21 architectures. More information is available in the 1.5.2 release announcement.
July 2001
2001-07-11: NetBSD 1.5.1 released
NetBSD 1.5.1 released with support for 21 architectures. More information is available in the 1.5.1 release announcement.
May 2001
2001-05-22: 1.5.1_BETA2 snapshot available
A NetBSD/x68k 1.5.1_BETA2 snapshot, based off of 1.5.1 sources as of 2001-05-19, is now available in the NetBSD/x68k snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5.1 was released on July 11th, 2001)
March 2001
2001-03-19: New 1.5.1_ALPHA snapshot available
A new 1.5.1_ALPHA snapshot based on 2001-03-14 sources is now available in the NetBSD/x68k snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5.1 was released on July 11th, 2001)
January 2001
2001-01-15: New snapshot
The NetBSD-current/x68k snapshot as of 2001-01-11 (1.5Q) is now available in the NetBSD/x68k snapshot area of the ftp server.
December 2000
2000-12-06: NetBSD 1.5 released
NetBSD 1.5 released with support for 20 architectures. More information is available in the 1.5 release announcement.

NetBSD/x68k is available in source form only for the NetBSD 1.5 release. NetBSD 1.5.1 will have release binaries for NetBSD/x68k.

November 2000
2000-11-25: NetBSD 1.4.3 released
NetBSD 1.4.3 released with support for 16 architectures. More information is available in the 1.4.3 release announcement.
July 2000
2000-07-25: First 1.4.3_ALPHA snapshot
The first x68k 1.4.3_ALPHA snapshot, based off of 1.4.3 sources as of 2000-07-16, is now available in the NetBSD/x68k snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.4.3 was released on November 25th, 2000)
April 2000
2000-04-29: Binary Packages
Over 600 MB of binary packages available for NetBSD 1.4.2 and later.

2000-04-28: Snapshot
New snapshot available.
March 2000
2000-03-19: NetBSD 1.4.2 released
NetBSD 1.4.2 released with support for 16 architectures.
January 2000
2000-01-14: ZS/MS fix
The implementation of ms(4) polling to use timeout(9) software timer, and change the clock interrupt handler not to lower interrupt level, in order to eliminate glitches on MPU interrupt levels.

2000-01-06: Snapshot
New release branch snapshot available.
October 1999
1999-10-02: Snapshot
New snapshot available.
July 1999
1999-07-09: Snapshot
New snapshot available. Includes KAME (IPv6), improvements to 060 support and fpu emulator, and memswitch(8).
June 1999
1999-06-22: memswitch
memswitch(8) utility added.
May 1999
1999-05-12: NetBSD-1.4 released
The NetBSD 1.4 release supports sixteen architectures, and was distributed free by Usenix to all 1999 ATC attendees.
March 1999
1999-03-18: Snapshot
New snapshot available.

1999-03-16: Bus drivers
Some drivers are bus.h'fied and reorganized under newly-introduced "intio" virtual device.
January 1999
1999-01-04: Snapshot
New snapshot with UVM and EGCS is available.
December 1998
1998-12-23
UVM is enabled by default.
November 1998
1998-11-14
Unofficial X binary sets including the server for 1.3.2 are available.

1998-11-11
X server is imported to the master source tree!!
September 1998
1998-09-02
The boot program is updated to support booting from the second, third,... disk.
August 1998
1998-08-07
The X clients can now be built for NetBSD/x68k.

1998-08-04
The ITE keymap files are installed into /usr/share/keymaps.
June 1998
1998-06-30
NetBSD-current can now be compiled for x68k machines again. UVM, the new, vastly improved virtual memory system, is optionally available for NetBSD/x68k.
