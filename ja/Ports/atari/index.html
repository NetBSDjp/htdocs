NetBSD/atari

NetBSD/atari について 

NetBSD/atari は、 Atari シリーズのパーソナルコンピューターへの NetBSD ポートです。 NetBSD/atari の開発活動は、開発者の余暇に応じたスピードで 進んでいます。現在、 NetBSD/atari は TT030、 Falcon および Hades で動作します。 Milan 向けの実験的なサポートもあります。

容易にインストールすることができる NetBSD/atari のバイナリー配布は、 1.5.2 リリースと NetBSD-current のスナップショットがあります。


NetBSD/atari ニュース 

2001-09-22: NetBSD/atari が ELF に移行
Thomas Gerner は、 NetBSD/atari ポートを ELF に変更する修正を コミットしました。 COMPAT_AOUT_M68K カーネルオプションがあれば、 すべての古い a.out バイナリーはそのまま動作するはずです。 a.out 用の ifconfig(8) と route(8) は、構造体の境界調整問題のせいで 動作しないことに注意してください。

NetBSD/atari ELF スナップショットは ftp サーバーの atari スナップショット 置き場にあります。また、このスナップショット、 またはソースからアップグレードするための 手順 もあります。

2001-09-13: NetBSD 1.5.2 リリース
21 のアーキテクチャーをサポートした NetBSD 1.5.2 がリリースされました。 より詳しい情報は 1.5.2 リリース声明にあります。

2001-07-11: NetBSD 1.5.1 リリース
21 のアーキテクチャーをサポートした NetBSD 1.5.1 がリリースされました。 より詳しい情報は 1.5.1 リリース声明にあります。

2001-06-12: 実験的な Milan カーネルが利用可能
実験的な Milan カーネルが、 ftp サーバーの NetBSD/atari スナップショット 置き場から入手できます。

対応ハードウェア 

最低限、システムには 68030 CPU、 4MB の RAM (うち 2MB は ST-RAM で可)と、 SCSI または IDE ディスクが必要です。配布されている BOOT および BOOTX カーネルはいずれも FPU エミュレーションを含んでいるので、 FPU は実際には必須ではありません。現状でのエミュレーションは mc68882 命令セットを完全にカバーしてはいませんが、非常に使えるシステムであると 認めていただけるでしょう。

NEW! 印の付いたシステムは NetBSD-current でのみサポートされます。それ以外のハードウェアは NetBSD 1.5 でサポートされています。

  • TT-HIGH を含む、 ST および TT ビデオモード
  • Falcon ビデオ (Direct Color - 15/16 bit depth は除く)
  • Hades et4000/w32-pci ビデオアダプター
  • Hades et6000-pci および et6100-pci ビデオアダプター
  • 内蔵 5380 SCSI アダプター
  • ほとんどの SCSI ディスク、 CD-ROM、テープおよび ZIP ドライブ
  • リアルタイムクロック
  • SCC シリアルポート (serial2/modem2)
  • 720Kb/1.44Mb フロッピードライブ
  • パラレルプリンター
  • Falcon および Hades 上の IDE インターフェース (ATAPI を含む)
  • ひとつ目の 68901 UART (modem1) 上のシリアルインターフェース
  • Hades における 68060 をサポート
  • Falcon FX メモリー拡張
  • atari マウス
  • 3 ボタンマウス ( 作り方参照)
  • サポートされている VME バスデバイス (TT030/Hades)
    • VME BVME410 イーサネット
    • Circad Leonardo 24-bit VME グラフィックアダプター
    • Crazy Dots VME et4000 グラフィックアダプター
    • VME Riebl (and possibly PAM) イーサネット
  • サポートされている Hades PCI デバイス
    • Adaptec 2940U SCSI NOT! ( 下記参照)。
    • ESS Technology Inc. Solo-1 サウンドカード
    • 3Com 3c59x ネットワークカード
  • サポートされている Hades ISA デバイス
    • teles 16.3 カードでの I4BSD サポート
    • NE2000 互換カード
  • サポートされている Milan PCI デバイス
    • Intel EtherExpress PRO 10+/100B NEW!

注意!

  • Hades PCI バスは非常にクリティカルです。ほとんどのカードは認識されません。 これは電気的な問題が原因のようです。
  • et4000 初期化コードのバグのため、古い NetBSD1.4 カーネルは、 et4000 を持つ Hades では、 画面を黒や白のブロックでちらつかせます。修正した カーネルがあります。また、 1.4.1 か 1.4.2 のカーネルを使ってもよいでしょう。
  • 現在の Adaptec ドライバーは、残念ながら、もはや動作しません。
