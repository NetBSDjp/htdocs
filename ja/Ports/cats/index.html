NetBSD/cats

About NetBSD/cats 

NetBSD/cats is an port of NetBSD to the Chalice Technology CATS motherboard.

Up until the release of NetBSD 1.6, this port used to be part of the NetBSD/arm32 port. As NetBSD 1.6 hasn't been released yet, these machines are supported in NetBSD/arm32 1.5.1.


NetBSD/cats News 

2001-06-10: Cats moves to it's own arch dir.
NetBSD/cats is now in it's own arch dir, sys/arch/cats Kernels should be built from there. Note that GENERIC has grown as it now contains all the currently supported USB devices. Any cats specific bugs should be reported using the newly created port-cats category in send-pr.
