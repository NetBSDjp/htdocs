NetBSD/x86_64

NetBSD/x86_64 is a port to AMD's upcoming 64 bit architecture. Currently it is only known to run on the Virtutech simulator (Simics), since the actual hardware is not available yet. It was committed to the NetBSD source tree on june 19th, 2001.

The work to do this port was done by Frank van der Linden at Wasabi Systems, assisted by AMD, who provided the simulator and the hardware on which to run it.

The current status is that it runs multi-user inside the simulator.
