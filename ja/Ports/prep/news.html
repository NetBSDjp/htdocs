NetBSD/prep news archive

February 2002
2002-02-26: Motorola Ultra 603 and Ultra Plus support
Platform support was extended to the Motorola Ultra 603 and Ultra Plus boards, which are the buildling blocks for systems such as the Bull ESTRELLA 200 series.
March 2001
2001-03-12: Linux compatibility enabled for PowerPC ports
Emmanuel Dreyfus has integrated Linux binary compatibility for PowerPC ports. This means that NetBSD/prep should now be able to run LinuxPPC binaries such as Netscape Communicator. More information is available in the Change section.
August 2000
2000-08-06: New snapshot available
Userland and kernel snapshot, based on 2000/08/02 sources, is now available in the NetBSD/prep snapshot area of the ftp server.
June 2000
2000-06-08: USB support
USB support added.

2000-02-29: Initial import
Initial import of prep port.
