NetBSD/prep

About NetBSD/prep 

NetBSD/prep is the port of NetBSD to early PReP (PowerPC Reference Platform) machines.


NetBSD/prep News 

2003-04-21: NetBSD 1.6.1 released
NetBSD 1.6.1 released with support for 40 architectures. More information is available in the 1.6.1 release announcement.

2003-01-16: IBM RS/6000 ThinkPad 820 6040 support
Platform support was extended to the IBM RS/6000 ThinkPad 820 6040.

2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.

2002-05-30: IBM RS/6000 43P 7043-140 support
Platform support was extended to the IBM RS/6000 43P 7043-140.

2002-05-05: New snapshot available
Userland and kernel snapshot, based on 2002/05/02 sources, is now available in the NetBSD/prep snapshot area of the ftp server.

2002-02-26: Motorola Ultra 603 and Ultra Plus support
Platform support was extended to the Motorola Ultra 603 and Ultra Plus boards, which are the buildling blocks for systems such as the Bull ESTRELLA 200 series.

2001-03-12: Linux compatibility enabled for PowerPC ports
Emmanuel Dreyfus has integrated Linux binary compatibility for PowerPC ports. This means that NetBSD/prep should now be able to run LinuxPPC binaries such as Netscape Communicator.
