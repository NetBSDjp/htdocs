2001-04-02: Hardware scrolling for RiscPC/A7000/NC wscons
Reinoud Zandijk has committed code for hardware scrolling for the RiscPC and compatibles. This new code has been tested in different screen resolutions and depths, and its all worked ok so far. It even appears to run faster than the original vidcconsole code. More details are available in Reinoud's announcement in the port-arm32 mail archive.

October 1999
1999-10-12: mpg123 package for arm32
Thanks to Simon Burge of the NetBSD Project, owners of Systems equipped with a StrongARM CPU can now enjoy playing MP3 files on their systems, too. The StrongARM CPU doesn't have a FPU, which was previously required by the mpg123 software to play sound. This limitation is now gone thanks to Simon's rewriting of the software to use fixed point (integer) math.
