NetBSD/arm32 の歴史

NetBSD/arm32 は、もともと RiscBSD と呼ばれるプロジェクトの 基本部分を構成するものでした。 RiscBSD グループ (もとは、 Mark Brinicombe、 Robert Black、 Neil Carson、 Manar Hussain、 Scott Stevens、 Melvin Tang-Richardson) は、 Acorn が新たに発表した ARM610 プロセッサーベースの RiscPC システムへ、フリー unix オペレーティングシステムを移植したいと 考えていました。 RiscBSD プロジェクトは、 Acorn の RiscPC 発売時に、 Xara の Gordon Taylor の参加のおかげで、はじめて形になりました。 それから、 Mach、 Linux、 FreeBSD などを慎重に研究したのち、 私たちは OS として NetBSD を選びました。

時間が経つにつれ、このチームは縮小し、 Mark、 Neil、 Robert、 Scott が残るだけになりました。その後、 Network Computer (現 Liberate Technologies) は、 Digital Shark コンピューター上での使用のために、オペレーティングシステムおよび関連 toolchain のさまざまな拡張に出資しました。

最近では、 NetBSD/arm32 ツリーはおもに人々の余暇を使って書かれています。 貢献や作者達についての詳細は、このポートのメイン web ページに一覧があります。 現在、 VLSI RC7500 ポートが実際に作業をしているかどうかはわかりません。 どなたか援助していただける方がいらしたら、大変ありがたいです。

最近では、 RiscBSD は単なる NetBSD/arm32 の Acorn マシン向けの CD ROM 配布となっており、 Mark Brinicombe が作っています。 最初の RiscBSD 'チーム' は、もう存在しません。 RiscBSD 配布についてより詳しくは、 RiscBSD web サイトを見てください。
