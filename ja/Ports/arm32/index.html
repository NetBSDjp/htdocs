NetBSD/arm32

NetBSD/arm32 について 

NetBSD/arm32 1.5.3 は、 NetBSD/arm32 の 9 回目の公式リリースです。

NetBSD/arm32 は、 ARM および StrongARM で動作するさまざまな コンピュータープラットフォーム向けのポートです。このポートは 1994 年から作業がおこなわれており、 Mark Brinicombe がポートメンテナーです。

「 arm32 」なるものは存在しません。最初の ARM プロセッサー (ARM2 と ARM3) は、 Acorn が設計し、 26 ビットの制限と貧弱な MMU を持っていました。これらのプロセッサーは NetBSD/acorn26 (かつては NetBSD/arm26 でした) でサポートされています。 のちに Acorn は Apple と VLSI とともに ARM を興しました。 ARM の CPU (6, 7, 8, 9 および StrongARM) は完全に 32 ビットで、 これらが NetBSD/arm32 でサポートされています。

その後、 Digital は、より速い CPU である StrongARM を生産するため、 ARM アーキテクチャーのライセンスを 受けました。この技術は Intel に移転され、現在ではこの CPU は Intel StrongARM になっています。

Ben Harris は、 EtherI や EtherN カードで使われている MX98905 チップのバグ回避のため "ne at podulebus" ドライバーを修正し、 これらのカードが NetBSD で使えるようになりました。彼はまた、 vidcvideo ドライバーにも手を加え、 ARM7500 でスクリーンモードを 正しく認識するようにしました。これらの結果、 NC_WSCONS がマルチユーザーで、ちゃんとしたディスプレイで動くようになりました。

Ben は今度は彼の Acorn NC で X を使うための作業をしています。

サポートされている ARM および StrongARM CPU 

ここに掲げるのは、 NetBSD/arm32 でのサポートがある CPU その他の部品です。 ここではビデオコントローラー、 core logic 、 PCI ブリッジ、 浮動小数点演算ユニットなどをリストしています。

  • ARM600, ARM610
  • ARM700, ARM710
  • ARM810
  • ARM7500FE
  • ARM VIDC
  • ARM FPA10, FPA11
  • ARM IOMD
  • Intel DC21285 ("footbridge")
  • Intel SA-110


サポートされているコンピューターシステム 

下記のシステムは NetBSD 1.5 では NetBSD/arm32 でサポートされていましたが、 NetBSD 1.6 では別ポートに分離される予定です。すでに NetBSD-current では分離されています。

  • Acorn Risc PC 600、 Risc PC 700、 StrongARM Risc PC、 J233、 A7000 および A7000+ は、 NetBSD/acorn32 でサポートされるようになりました。
  • Digital DNARD ("Shark") は、 NetBSD/shark でサポートされるようになりました。
  • Chalice Technology CATS は、 NetBSD/cats でサポートされるようになりました。

初期の ARM ベースの Acorn マシンは NetBSD/acorn26 (かつては NetBSD/arm26 でした) ポートでサポートしています。 ARM ベースの Windows CE PDA は NetBSD/hpcarm でサポートしています。 StrongARM ベースの NetWinder マシンは、 NetBSD/netwinder でサポートしています。


人々 

コアプロセッサーのサポートは Mark Brinicombe (その大部分) と Neil Carson (libkern 、 pmap 、浮動小数点エミュレーター、 ライブラリーのいくつか) によって書かれました。

Acorn プラットフォームのサポートは、 Mark Brinicombe (その大部分) によって、 Scott Stevens (さまざまなデバイスドライバー) 、Melvin Tang Richardson (VIDC コンソール) 、 Mike Pumford (EtherH ドライバー) および Andrew McMurry (Acorn FileCore ファイルシステム) の協力を受けて、 書かれました。

Digital Shark のサポートは、 Dave Chaiken、 Chris Demetriou、 Jay Kistler、 Todd Vierling らによって書かれました。 21285 と CATS のサポートは、 Mark Brinicombe によって書かれました。 VLSI RC7500 のサポートは、 VLSI の Danny Tsen から寄贈されました。


進行中の作業 

私たちは現在、新しい機種独立なコンソールの統合、 pmap のアップグレード、 Acorn マシン用 XFree86 サーバーの作成、全般的なコードのクリーンアップ およびさらなる性能の改良をしているところです。

私たちは、皆さんに現在の NetBSD/arm32 問題報告の検証をしていただきたいと思います。
