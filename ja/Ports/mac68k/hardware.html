NetBSD/mac68k Supported Hardware

NetBSD is as useful and as good as its users make it.

If you are interested in helping with something, take a look at the list of things to do and choose something!

NetBSD works with a growing number of the older Macs. Take a look at the Current Information page or join the port-mac68k mailing list for the most current information. Also check out the machine-status document and the user survey.

Finally, the NetBSD/mac68k Operating System Information document provides a list of working hardware and software, as well as descriptions of known bugs and common problems.

What about the PowerMacs?

Although they share a few bits of hardware, the PowerMacs are a completely different architecture than their 68k cousins, and you can not run NetBSD/mac68k on them. The 68k-emulator does emulate the 68k processor very well for MacOS applications, but it is not sufficient to run NetBSD/mac68k for several reasons.

Fortunately, there is now a native port of NetBSD that will run on many of the PowerMac models. Please see the NetBSD/macppc port page for more details.
