9.2 MacLinux

最近 Apple は Linux の PowerPC Macintosh への移植版への支持を発表しま した。詳しくはこちらを: http://www.mklinux.apple.com/

更に MacLinux/68k プロジェクトというものがあります。どちらかと言えば何 事もゆっくりと進んでいるようではありますが。詳しい情報については次の URL をご覧ください。

http://maclinux.ml.org/

[訳註: NetBSD には Linux エミュレーションレイヤというものがありま すが、同じ Linux でも CPU アーキテクチャが異るとシステムコール番号が異 る、という問題があるので、NetBSD/m68k で Linux/m68k エミュレーションを 動作させることは、現時点では恐らく無理でしょう。]

9.3 MacMiNT

MacMiNT との互換性はありません。 MacMiNT は以下の URL で見つかると思います:

ftp://atari.archive.umich.edu

9.4 MacMach

残念ながら、MacMach [訳註: CMU Mach マイクロカーネルの Macintosh バージョン]は何年も前にプロジェクトが止まってしまいました。しかし、 PowerMac Linux が Mach マイクロカーネルの上に構築されています。OSF の OSF マイクロカーネ ル上の Linux に詳細があります。

9.5 A/UX

A/UX ファイルシステムは read-only でマウントできます。将来は A/UX バイ ナリ互換機能が提供されるかも知れません(あまり期待しないように)。

9.6 MacOS

Lawrence Kesteloot の hfs ユーティリティが、 MacOS ファイルシステムを read-only でアクセスするのに使えます。このツ ールで MacOS ファイルシステムをブラウズしたり、NetBSD/mac68k パーティ ションへファイルをコピーしたりするのに使ってください。いつか MacOS が NetBSD/mac68k 上で１プロセスとして動作する日が来るのを期待しましょう。

hfs 以外にも、 hfsutils とい うユーティリティ群が発表されており、Mac HFS ボリュームを NetBSD から参 照するのに使えます。
