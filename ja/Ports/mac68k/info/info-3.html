3. 既知のバグとパッチ情報

NetBSD/mac68k の最新バグリストは、 ftp://ftp.netbsd.org/pub/NetBSD/NetBSD-1.3/BUGS から得られます。

3.1 ADB

  • たくさんの 3-ボタンマウスが、未だにカーネルのドライバでサポートされていません。

3.2 Ethernet

  • まだすべての 8390 および SONIC カードが対応されてはいません。

3.3 シリアル

  • たまに fifo/リングバッファオーバーランを起こすことがまだあります。

3.4 PRAM

  • 正確な時刻を書き込めないでしょうが、NetBSD/mac68k での時計の遅れのことを考えれば、たぶんその方が良いのでは。

3.5 カーネル

  • FPU のエミュレートが必要 − FPE はほとんどできてます。かなりの FPU なしの機種も現在ではブートします(すべての current カーネルは FPE が組み込んであり、FPU-less math ライブラリを使えばまだエミュレートされ ない命令を避けることができます)。
  • 'LC040 マシンでセグメンテーションフォールトをたくさん起こす FPE の問題。
  • 実際に何か動いていると時計が遅れていく問題(ちょっとましにはなった)。
  • vmstat 統計/カーネルのメモリ使用量がちょっと多いような(?)

3.6 Mac ユーティリティ

Booter

  • 大量のクリーンアップ
  • ネイティブブート (つまり MacOS なしで) ?
  • 1.9.4 より後で 1.10.3 より前のバージョンは IIvx で正常に動作しな いようです。1.10.3 以降では修正されています。

Installer

  • 新ファイルシステム用に更新
  • "fstat: file not found" エラー (無害なので無視してください)
  • 1.1 バージョンは FPU がない Mac ではクラッシュすることがある問題。1.1d 以降を使ってください。
  • アップデートの際、GMT ではなくローカルタイムゾーンを使用する問題。1.1e 以降では修正。
  • Build Devices を実行するとデフォルトでは 4 個の pty しか作られない問題。1.1e 以降では修正。
  • MiniShell でのページングが使用できない問題。1.1f 以降では修正。
  • 余分の pty が不正に作られる問題。1.1f 以降では修正。

Mkfs -> Newfs

  • SCSIError#5 メッセージを無視する
  • Newfs に改名 (レベル 2 FFS 対応がまず必要)
  • Installer にマージする
