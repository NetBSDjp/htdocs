2002-07-22: NetBSD 1.5.3 released
NetBSD 1.5.3 released with support for 20 architectures. More information is available in the 1.5.3 release announcement.
September 2001
2001-09-13: NetBSD 1.5.2 released
NetBSD 1.5.2 released with support for 21 architectures. More information is available in the 1.5.2 release announcement.
July 2001
2001-07-11: NetBSD 1.5.1 released
NetBSD 1.5.1 released with support for 21 architectures. More information is available in the 1.5.1 release announcement.
May 2001
2001-05-13: 1.5.1_BETA2 snapshot available
A NetBSD/vax 1.5.1_BETA2 snapshot, based off of 1.5.1 sources as of 2001-05-09, is now available in the NetBSD/vax snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5.1 was released on July 11th, 2001)

2001-05-06: Two new drivers committed
Anders Magnusson has committed a driver for the Interlan NI1010 Ethernet Controller (il) and a driver for the DEC DDCMP DMC-11/DMR-11 interface (dmc) to NetBSD/vax -current. Both drivers are from 4.4BSD.

2001-05-01: Support for VAX 4000/705A
Michael Kukat has written support for the VAX 4000/705A.
April 2001
2001-04-28: VAX-specific in4_cksum() function repaired
Anders Magnusson has repaired the VAX-specific in4_cksum() function, which, in turn, will improve performance. For example, a VAX3900 with DEQNA (the bad one) peeks just above 500K/s now, around 50% of the cable capacity, which is actually very good.

2001-04-09: 1.5.1_BETA snapshot available
A NetBSD/vax 1.5.1_BETA snapshot, based off of 1.5.1 sources as of 2001-04-08, is now available in the NetBSD/vax snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5.1 was released on July 11th, 2001)
February 2001
2001-02-04: Support for new CPU types
Michael Kukat has written support for a bunch of new CPU types, among them are MicroVAX 3100m90, MicroVAX 3100m95, VAX 4000/100, VAX 4000/105A, VAX 4000/108, VAX 4000/400, VAX 4000/500 and VAX 4000/600.

November 2000
2000-11-25: NetBSD 1.4.3 released
NetBSD 1.4.3 released with support for 16 architectures. More information is available in the 1.4.3 release announcement.

2000-11-07: 1.5_BETA2 snapshot
A NetBSD/vax 1.5_BETA2 snapshot is now available in the NetBSD/vax snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5 was released on December 6th, 2000)
October 2000
2000-10-24: 1.5_BETA snapshot
A NetBSD/vax 1.5_BETA snapshot is now available in the NetBSD/vax snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5 was released on December 6th, 2000)
September 2000
2000-09-12: First 1.5_ALPHA2 snapshot
The first NetBSD/vax 1.5_ALPHA2 snapshot is now available in the NetBSD/vax snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5 was released on December 6th, 2000)
July 2000
2000-07-14: Support for dynamically linked VAX ELF programs
Matt Thomas has committed changes so that ld.elf_so can now load and run dynamically linked VAX ELF programs.
June 2000
2000-06-26: Regained support for MFM disks on VS2000
Anders Magnusson has checked in fixes to use the VS2000 MFM disks again. Along with that, VS2000 SCSI DMA should be able to cooperate with it. VS2000 still doesn't have floppy support back yet, since there is no obvious way to detect which type of drive there is. More details are available in Ander's announcement in the port-vax mail archive.

2000-06-22: First 1.5_ALPHA snapshot
The first vax 1.5_ALPHA snapshot, based off of 1.5 sources as of 2000-06-20, is now available in the NetBSD/vax snapshot area of the ftp server. (snapshot deprecated. NetBSD 1.5 was released on December 6th, 2000)

2000-06-10: VAX 8200 Multiprocessor support
Spinup code for the slave CPUs on a VAX8200 multiprocessor system added by Anders Magnusson.
May 2000
2000-05-18: New snapshot available
A new snapshot based on 2000-05-17 sources is available at: ftp://ftp.NetBSD.org/pub/NetBSD/arch/vax/snapshot/.

2000-05-08: VAX 4000/200 support
Support for VAX 4000/200 added, written by Michael Kukat.
April 2000
2000-04-22: Support for RL01/RL02 disks
Anders Magnusson has written a driver for the DEC RL01/RL02 disks.

2000-04-13: Support for SCSI on VS4000
Matt Thomas has written an attachment and support code for the NCR 53C94 SCSI chip used on the VAXstation 4000. It fully uses the DMA capabilities of the VS4000.

2000-04-12: Support for DEBNA/DEBNT/DEBNK
Anders Magnusson has written a driver for the BI bus ethernet cards DEBNx.
March 2000
2000-03-19: NetBSD 1.4.2 released
NetBSD 1.4.2 released with support for 16 architectures.
January 2000
2000-01-20: Interrupt Dispatching Reworked
For the first time since BSD 4.2, the VAX interrupt dispatch code has been reworked. Instead of a small integer (0..63), a full 32 bit value be passed to an interrupt routine.
June 1999
1999-06-25: Precompiled packages
Over 100 precompiled packages and X clients available on ftp.NetBSD.org. Many thanks to Brian Chase.

1999-06-06: VAX 4000/300 support
Support for VAX 4000/300 added, written by Michael Kukat.
December 1998
1998-12-01
New snapshot available.
October 1998
1998-10-31
Matt Thomas has committed the final changes to enable shared libraries on the VAX port.
August 1998
1998-08-21
The page size is increased from 1k to 4k. This gives a noticeable speedup of the whole system.

1998-08-10
First cut of support for the VS4000 model 60.
July 1998

NetBSD/vax is now using the new faster pmap interface (PMAP_NEW) as the default, as well as UVM.
June 1998

Support for the monochrome graphic console on VS2000 and VS3100. No X yet, though.
