NetBSD/newsmips

NetBSD/newsmips について 

NetBSD/news68k は、 SONY の m68k ベースの "NET WORK STATION" シリーズ向けの NetBSD ポートです。これらのマシンの 追加情報を用意しました。

NetBSD/newsmips ポートは、 NetBSD/{mips,pmax} と 4.4BSD Lite2 をもとにしています。

現在のところ、まだ "実験的ポート" です。

m68k ベースの NEWS マシンは、 NetBSD/news68k ポートで対応しています。


NetBSD/newsmips ニュース 

2001-04-12: 新しい -current スナップショット
2001-04-02 現在のソースに基づく新しい -current のスナップショットが ftp サーバーの NetBSD/newsmips スナップショット 置き場から入手できます。

2000-12-06: NetBSD 1.5 リリース
20 のアーキテクチャーをサポートする NetBSD 1.5 をリリースしました。 より詳しい情報は 1.5 リリース声明にあります。

2000-11-13: MI wscons 対応
NetBSD/newsmips は MI wscons に以降しました。現在、 NWS-3470D 上の NWB-253 と、 NWS-5000 の "XA" フレームバッファーに対応しています。

2000-10-30: NWS-5000 の SCSI に対応
news5000 で使われている HP SPIFI3-SE SCSI インターフェースへの対応が 加わりました。

2000-10-12: tulip ベースの NWB-5852A に対応
尾上淳氏によって、 tulip ベースの 100BaseTX カード、 NWB-5852A への対応が加わりました。これは DEC-21140A チップを使った APbus ボードです。 100BaseTX のみに対応しています。 Half duplex/Full duplex は手動で切り替えられますが、 オートネゴシエーション機能はありません。
NetBSD/newsmips ニュース記事のアーカイブ


対応機種 

現在のところ、 NetBSD/newsmips はシングル CPU の機種のみに対応しています。

  • R3000 と hyper-bus ベースの機種:
    • NWS-3470D
    • NWS-3410
    • NWS-3720

  • R4000/4400 と apbus ベースの機種:
    • NWS-5000


対応周辺機器 

  • オンボード LANCE イーサネットインターフェース (le)
  • オンボード SONIC イーサネットインターフェース (sn)
  • オンボード SONY CXD1185 SCSI インターフェース (sc)
  • オンボード HP SPIFI3-SE SCSI インターフェース (spifi)
  • ほとんどの SCSI デバイス
  • オンボード Z8530 シリアルインターフェース (zsc)
  • キーボードとマウス
  • NWB-253 フレームバッファー (テキストのみ)
  • news5000 の "XA" フレームバッファー (テキストのみ)
  • NWB-5852A 100BaseTX イーサネットインターフェース (tlp)


未対応のもの 

機種 / デバイス / 機能:

  • IOP ベースの機種 (NWS-3800 シリーズ)
  • R4600 ベースの機種 (NWS-4000 シリーズ; 作業中)
  • 内蔵フロッピーディスク
  • 上記以外のフレームバッファー
  • X サーバー
  • COMPAT_NEWSOS
  • その他いろいろ
