NetBSD/newsmips news archive

September 2002
2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.
April 2002
2002-04-20: New -current snapshot available
A new -current snapshot based on 2002-04-17 sources is now available in the NetBSD/newsmips snapshot area of the ftp server.

February 2002
2002-02-19: New -current snapshot available
A new -current snapshot based on 2002-02-10 sources is now available in the NetBSD/newsmips snapshot area of the ftp server.

April 2001
2001-04-12: New -current snapshot available
A new -current snapshot based on 2001-04-02 sources is now available in the NetBSD/newsmips snapshot area of the ftp server.

November 2000
2000-11-13: MI wscons support
NetBSD/newsmips has switched to MI wscons. Currently NWB-253 on NWS-3470D and "XA" framebuffer on NWS-5000 are supported.
October 2000
2000-10-30: Support for SCSI on NWS-5000
Support for the HP SPIFI3-SE SCSI interface used on news5000 is added.

2000-10-12: Support for tulip based NWB-5852A
Atsushi Onoe has added support for the tulip based 100BaseTX card NWB-5852A, which is an APbus board with DEC-21140A chip. It supports 100BaseTX only. Half duplex/Full duplex can be specified manually, but there is no auto negotiation functionality.
December 1999
1999-12-22: Support NWS-5000 series
NWS-5000 support is mostly done by SHIMIZU ryo.
July 1999
1999-07-08: Native boot loader added
Native boot loader has been added to NetBSD/newsmips.

1999-07-06: New NetBSD/newsmips page available.
Many thanks to Izumi Tsutsui.
