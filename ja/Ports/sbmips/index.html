NetBSD/sbmips

About NetBSD/sbmips 

NetBSD/sbmips is a new port of NetBSD to Broadcom's SiByte Boardband Processor.

More details will show up here shortly.


NetBSD/sbmips News 

2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.


Supported System Models 

It currently supports the

  • BCM91250A (Swarm) evaluation board with the BCM1250 CPU.
