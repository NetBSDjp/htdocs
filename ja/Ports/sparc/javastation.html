NetBSD/sparc JavaStation Support

Patching NetBSD-current

JavaStation support is still "experimental" and requires patches that are not in the main source for the NetBSD kernel. Contact Valeriy E. Ushakov for the current status of JavaStation support and for the latest patches.

JavaStation-1 (a.k.a. "Mr.Coffee") Support:

As mentioned in the FAQ, there are two types of JavaStation-1 units -- those with OpenBoot PROM 2 (OBP2), and those with Open Firmware (OFW). Which type of boot ROM you have affects how you will boot your system and what devices are supported.

The banner the JavaStation displays at boot-time indicates which is present. If it displays "OpenBoot 2.30", it has OpenBoot PROM 2. If it displays "OpenBoot 3.x" (where x is usually 10 or 11), it has Open Firmware.

Unfortunately, the OpenBoot PROM 2 does not properly initialize the PS/2 style keyboard/mouse controller. This means you can't configure the PROM without a serial console. The JavaStation kernel patches support PS/2 keyboards, so you can use the keyboard once the NetBSD kernel is loaded.

Open Firmware has no PS/2 keyboard/mouse issues.

The Xsun X11R6 server works with the PS/2 keyboard and mouse (kernel patches not committed).

Audio playback and recording work.

OpenBoot PROM 2-based JavaStation-1 units will netboot like any other NetBSD/sparc machine. See the sparc-specific section of the Diskless NetBSD HOW-TO.

Open Firmware-based JavaStation-1 units will netboot like any other modern machine -- they send a DHCP request for a TFTP server and bootloader filename, then download and execute that file. These units require a different bootloader than other NetBSD/sparc machines. Make sure you use bootjs.net from the snapshot or distribution. See the javastation-specific section of the Diskless NetBSD HOW-TO.

JavaStation-10 (a.k.a. JavaStation-NC or "Krups") Support

As mentioned in the FAQ, all JavaStation-10 units have Open Firmware.

Open Firmware has no PS/2 keyboard/mouse issues.

There is support for text console on these units.

While the XFree86 server supports the framebuffer chipset used in these units, the chips are located on the PCI bus, which is problematic since no other NetBSD/sparc models have PCI busses. It should be possible to eventually support X on these units.

Audio playback and recording work.

JavaStation-10 units will netboot like any other modern machine -- they send a DHCP request for a TFTP server and bootloader filename, then download and execute that file. These units require a different bootloader than other NetBSD/sparc machines. Make sure you use bootjs.net from the snapshot or distribution. See the javastation-specific section of the Diskless NetBSD HOW-TO.
