NetBSD/evbmips

About NetBSD/evbmips 

NetBSD/evbmips is intended to be a port of NetBSD to various MIPS-based evaluation boards.

Supported System Models 

  • It currently supports the MIPS Malta evaluation board with both the 4Kc (MIPS32) CPU board and the 5Kc (MIPS64) CPU board (running in 32-bit mode only).
