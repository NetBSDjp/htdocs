May 2000
2000-05-20: Recall of AC Adapters for IBM Workpad z50.
IBM is voluntarily recalling AC Adapters for z50. Check your z50's S/N and FRU P/N. IBM will replace the AC Adapter, free of charge. IBM also requests that you do not leave your current AC Adapter plugged into any AC power outlet while unattended.

2000-05-07: Experimental X server for the hpcmips.
xsrc/xc/programs/Xserver/hw/netbsd/hpc is available. 8bpp StaticColor and 16bpp TrueColor are both supported. 2bpp gray scale is not supported.

2000-05-04: RAW keyboard mode support
Raw keyboard mode support has been added to the kernel (the same as NetBSD/i386). Key codes are compatible with that of a PC keyboard.

March 2000
2000-03-16: Binary snapshot of "NetBSD 1.4U" is available.
Userland binaries are compiled with "-mhard-float" flag. Older binaries are not binary compatible with this snapshot.
February 2000
2000-02-25: Cassiopeia E-series support will come next
Koji Suzuki has made Cassiopeia E-series CF test code. It works fine, and after some more testing, it will be committed to the current-tree.

2000-02-24: Slashdot article
A slashdot article titled Inexpensive Linux/BSD Handhelds says: 'a successful NetBSD Port means that this device could be the first widespread *NIX handheld'. Cheers!

2000-01-04: MGL2 support coming soon!
MGL2 for NetBSD/hpcmips soon will be available. On MC-R500, mgterm is working.
December 1999
1999-12-18: Presentation at InternetWeek 99
There was a short presentation on about NetBSD/hpcmips at the InternetWeek 99 BSD na hitotoki BOF in Yokohama, Japan.
November 2000
1999-11-30: TX3912/22 CPU support added.
Support for Toshiba TX3912/22 CPUs has been added. It is enabled with the kernel option ENABLE_MIPS_TX3900.

1999-11-26: port-hpcmips mailing list available.
The port-hpcmips@netbsd.org mailing list is available for discussion of NetBSD/hpcmips.

1999-11-23: New binary snapshot
A new binary snapshot is available. Read the installation notes for more information. (This snapshot is now obsolete, and is no longer available.)

1999-11-22: Supported Machine and Processor list contributed
Tim Rightnour has created a list of machines and processors supported by NetBSD/hpcmips, so that users can easily see if their systems are supported.

1999-11-18: Kernel floating point emulation merged.
Kernel emulation for floating point operations (written by Shuichiro URATA) has been merged. Now NetBSD/pmax binaries and packages work on NetBSD/hpcmips!

1999-11-13: Killer App demonstrated!
NetHack on wscons was demonstrated during a presentation at Open Source Matsuri in Akihabara, Japan.

1999-11-02: New console fonts available.
The FONT_VT220L8x8 and FONT_VT220L8x10 console fonts are now available.
October 1999
1999-10-30: Boot loader supports TX3922.
Support for the Toshiba TX3922 processors has been added to the boot loader.

1999-10-24: wscons driver now available.
The machine-independent wscons console driver now functions on NetBSD/hpcmips.

1999-10-10: First binary snapshot released!
The first NetBSD/hpcmips binary snapshot is available. (This snapshot is now obsolete, and is no longer available.)
September 1999
1999-09-25: If you build it, it will run!
The NetBSD/hpcmips kernel sources are in the NetBSD CVS repository, can be compiled, and work!

1999-09-17: NetBSD/hpcmips merge started.
Developers have started merging the NetBSD/hpcmips sources into the NetBSD CVS repository.
