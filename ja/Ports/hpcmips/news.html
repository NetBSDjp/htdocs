NetBSD/hpcmips news archive

  • Recent news (on port homepage)
  • Commits affecting the port-hpcmips portion of the source tree

Nov 26, 1999: port-hpcmips mailing list available.
The port-hpcmips@netbsd.org mailing list is available for discussion of NetBSD/hpcmips.

Nov 23, 1999: New binary snapshot
A new binary snapshot is available. Read the installation notes for more information. (This snapshot is now obsolete, and is no longer available.)

Nov 22, 1999: Supported Machine and Processor list contributed
Tim Rightnour has created a list of machines and processors supported by NetBSD/hpcmips, so that users can easily see if their systems are supported.

Nov 18, 1999: Kernel floating point emulation merged.
Kernel emulation for floating point operations (written by Shuichiro URATA) has been merged. Now NetBSD/pmax binaries and packages work on NetBSD/hpcmips!

Nov 13, 1999: Killer App demonstrated!
NetHack on wscons was demonstrated during a presentation at Open Source Matsuri in Akihabara, Japan.

Nov 02, 1999: New console fonts available.
The FONT_VT220L8x8 and FONT_VT220L8x10 console fonts are now available.

Oct 30, 1999: Boot loader supports TX3922.
Support for the Toshiba TX3922 processors has been added to the boot loader.

Oct 24, 1999: wscons driver now available.
The machine-independent wscons console driver now functions on NetBSD/hpcmips.

Oct 10, 1999: First binary snapshot released!
The first NetBSD/hpcmips binary snapshot is available. (This snapshot is now obsolete, and is no longer available.)

Sep 25, 1999: If you build it, it will run!
The NetBSD/hpcmips kernel sources are in the NetBSD CVS repository, can be compiled, and work!

Sep 17, 1999: NetBSD/hpcmips merge started.
Developers have started merging the NetBSD/hpcmips sources into the NetBSD CVS repository.

Aug 21, 1999: Initial demonstration with serial console.
Tetris on the serial console was demonstrated at a JNUG meeting.

Jan 12, 1999: pbsdboot and test.elf are available.

Jan 06, 1999: pbsdboot appears.
