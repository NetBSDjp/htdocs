NetBSD/news68k

NetBSD/news68k について 

NetBSD/news68k は、 SONY の m68k ベースの "NET WORK STATION" シリーズ向けの NetBSD ポートです。これらのマシンの 追加情報を用意しました。

NetBSD/news68k ポートは、NetBSD/newsmips と NetBSD/m68k ポートをもとにしています。

NetBSD 1.5 は、 NetBSD/news68k の最初の公式リリースです。


NetBSD/news68k ニュース 

2001-04-10: 1.5.1_BETA スナップショット
2001-04-06 現在の 1.5.1 のソースをもとにした NetBSD/news68k 1.5.1_BETA スナップショットが、 ftp サーバーの NetBSD/news68k スナップショット 置き場から入手できます。

2001-03-19: NetBSD/news68k が ELF に移行
NetBSD/news68k は -current において、ネイティブの オブジェクトフォーマットとして ELF を使うようになりました。 ELF スナップショットが ftp サーバーの NetBSD/news68k スナップショット 置き場から入手できます。このスナップショットやソースからの アップグレードの 手引きもあります。

2001-03-02: -current の新しいスナップショット
2001-02-16 現在のソースをもとにした -current の新しいスナップショットが、 ftp サーバーの スナップショット置き場 から入手できます。

2000-12-06: NetBSD 1.5 リリース
20 のアーキテクチャーをサポートする NetBSD 1.5 をリリースしました。 より詳しい情報は 1.5 リリース声明にあります。

対応機種 

現在のところ、 NetBSD/news68k は mc68030 ベースの シングルプロセッサーの機種のみに対応しています。

  • NWS-1750
  • PWS-1560
  • NWS-1250
  • NWS-1450
  • NWS-1460
  • NWS-1530
  • NWS-1580


対応周辺機器 

  • オンボード LANCE イーサネットインターフェース (le)
  • オンボード SONY CXD1180 SCSI インターフェース (si)
  • ほとんどの SCSI デバイス
  • オンボード Z8530 シリアルインターフェース (zsc)
    (シリアルコンソールにのみ対応)


未対応のもの 

機種 / デバイス / 機能:

  • mc68020 ベースのシステム (NWS-700/800)
  • IOP ベースのシステム (NWS-800/1800)
  • NWS-1200 の CXD1185 SCSI インターフェース
  • 内蔵フロッピーディスク
  • キーボード、マウスとフレームバッファー (作業中)
  • COMPAT_NEWSOS
  • その他いろいろ
