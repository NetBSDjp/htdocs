NetBSD/news68k

NetBSD/news68k について 

NetBSD/news68k は、 SONY の m68k ベースの "NET WORK STATION" シリーズ向けの NetBSD ポートです。これらのマシンの 追加情報を用意しました。

NetBSD/news68k ポートは、NetBSD/newsmips と NetBSD/m68k ポートをもとにしています。

NetBSD 1.6 は、 NetBSD/news68k の 5 回目の公式リリースです。

2001-03-19: NetBSD/news68k が ELF に移行
NetBSD/news68k は -current において、ネイティブの オブジェクトフォーマットとして ELF を使うようになりました。 ELF スナップショットが ftp サーバーの NetBSD/news68k (スナップショットは 消去されました。NetBSD 1.6 が 2002-09-14 に リリースされました) 。このスナップショットやソースからの アップグレードの 手引きもあります。

対応機種 

現在のところ、 NetBSD/news68k は mc68030 ベースの シングルプロセッサーの機種のみに対応しています。

  • NWS-1750
  • PWS-1560
  • NWS-1250
  • NWS-1450
  • NWS-1460
  • NWS-1530
  • NWS-1580
  • PWS-1550


対応周辺機器 

  • オンボード LANCE イーサネットインターフェース (le)
  • オンボード SONY CXD1180 SCSI インターフェース (si)
  • ほとんどの SCSI デバイス
  • キーボード (kb)、マウス (ms)
  • オンボード Z8530 シリアルインターフェース (zsc)
    (動作するフレームバッファーがまだないため、シリアルコンソールにのみ対応)


未対応のもの 

機種 / デバイス / 機能:

  • mc68020 ベースのシステム (NWS-700/800)
  • IOP ベースのシステム (NWS-800/1800)
  • NWS-1200 の CXD1185 SCSI インターフェース
  • 内蔵フロッピーディスク
  • フレームバッファー (作業中)
  • COMPAT_NEWSOS
  • その他いろいろ
