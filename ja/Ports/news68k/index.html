NetBSD/news68k

NetBSD/news68k について

NetBSD/news68k は、 SONY の m68k ベースの "NET WORK STATION" シリーズ向けの NetBSD ポートです。 これらのマシンの追加情報 を用意しました。

NetBSD/news68k ポートは、NetBSD/newsmips と NetBSD/m68k ポートをもとにしています。

NetBSD 1.5 は、 NetBSD/news68k の最初の公式リリースです。

対応機種
現在のところ、 NetBSD/news68k は mc68030 ベースの シングルプロセッサーの機種のみに対応しています。
  • NWS-1750
  • PWS-1560
  • NWS-1250
  • NWS-1450
  • NWS-1460
  • NWS-1530
対応周辺機器
  • オンボード LANCE イーサネットインターフェース (le)
  • オンボード SONY CXD1180 SCSI インターフェース (si)
  • ほとんどの SCSI デバイス
  • オンボード Z8530 シリアルインターフェース (zsc)
    (シリアルコンソールにのみ対応)
未対応の機種・デバイス・機能
  • mc68020 ベースのシステム (NWS-700/800)
  • IOP based systems (NWS-800/1800)
  • NWS-1200 のCXD1185 SCSI インターフェース
  • 内蔵フロッピーディスク
  • キーボード、マウスとフレームバッファー (作業中)
  • COMPAT_NEWSOS
  • その他いろいろ
