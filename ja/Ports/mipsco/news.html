NetBSD/mipsco news archive

September 2002
2002-09-14: NetBSD 1.6 released
NetBSD 1.6 released with support for 39 architectures. More information is available in the 1.6 release announcement.
May 2001
2001-05-25: New -current snapshot available
A new -current snapshot based on NetBSD 1.5V is available in the NetBSD/mipsco snapshot area of the ftp server.

This release includes an installation kernel with the NetBSD sysinst utility to simplify installations.

November 2000
2000-11-14: Model number information page
Model number information added to NetBSD/mipsco web pages.
October 2000
2000-10-07: New -current snapshot available
A new -current snapshot based on 2000-10-04 sources is now available in the NetBSD/mipsco snapshot area of the ftp server.
