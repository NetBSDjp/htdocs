NetBSD/mipsco

NetBSD/mipsco について 

NetBSD/mipsco は MIPS Computer Systems Inc. ファミリーの ワークステーションおよびサーバーへの NetBSD ポートです。

現在は Mips Magnum 3000 シリーズのマシンだけをサポートしていますが、 将来は他の機種もサポートしたいと考えています。

Mips Magnum 3000 は、 Bull や Olivetti といった会社によって 自社製品として再販されたポピュラーなワークステーションでした。

より新しい、 ARC 仕様の mips マシンは NetBSD/arc ポートでサポートされています。


NetBSD/mipsco ニュース 

2002-09-14: NetBSD 1.6 リリース
39 のアーキテクチャーをサポートした NetBSD 1.6 がリリースされました。 より詳しい情報は 1.6 リリース声明にあります。

2001-05-25: 新しい -current スナップショット
NetBSD 1.5V に基づく新しい -current のスナップショットが ftp サーバーの NetBSD/mipsco スナップショット置き場から入手できます。

このリリースには、インストールが簡単にできる、 NetBSD sysinst ユーティリティーを伴ったインストールカーネルが含まれています。

2000-11-14: 型番情報ページ
型番情報 が NetBSD/mipsco web ページに追加されました。

2000-10-07: 新しい -current スナップショット
2000-10-04 現在のソースに基づく新しい -current のスナップショットが ftp サーバーの NetBSD/mipsco スナップショット 置き場から入手できます。
