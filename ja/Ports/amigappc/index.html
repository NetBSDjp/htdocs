NetBSD/amigappc

NetBSD/amigappc is a very experimental port to the PowerPC-based Amiga expansion boards as manufactured by Phase 5.

Supported peripheral devices will basically be the same as with NetBSD/amiga. We do not plan to create a NetBSD-1.6 binary release, but development will continue and is in fact, easier, now that cross-compilation of the base system is supported within NetBSD-current.

A recipe to create a NetBSD/AmigaPPC kernel and a minimal system to test it will appear here shortly.
