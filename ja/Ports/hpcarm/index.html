NetBSD/hpcarm

NetBSD/hpcarm について

NetBSD/hpcarm は Intel StrongARM ベースの Windows CE PDA マシンで動作する NetBSD オペレーティングシステムです。 現在 SA1110 プロセッサーに対応しています。

"hpcarm" というポート名は、 NetBSD/hpcarm が動作する ハードウェア: マイクロソフト社の H/PC (ハンドヘルド PC) 仕様に準拠している StrongARM ベースのシステム、に由来します。 H/PC マシンはキーボードとタッチスクリーン、そして 8MB か それ以上の RAM を備えています。このポートは H/PC のほかに、 H/PC Pro と PsPC (パームサイズ PC) にも対応しています。 (多くの PsPC マシンが開発者達の実際の開発に使われています)

このハードウェアの大きさからして、 モバイルコンピューティング用アプリケーションを開発するときに NetBSD/hpcarm は良いプラットフォームとなるでしょう。 もちろん、 NetBSD/hpcarm は IPv6 や IPsec といった NetBSD の 標準的な機能の全てに対応しています。

対応システム

NetBSD/hpcarm は多くの種類の H/PC, H/PC Pro や PsPC マシンに対応しています。 あなたのシステムが対応しているかどうかは 対応システム 表 をご覧下さい。

対応プロセッサー

NetBSD/hpcarm は StrongARM プロセッサーに対応しています。 あなたのプロセッサーが対応しているかどうかは プロセッサー比較表 をご覧下さい。
