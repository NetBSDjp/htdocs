NetBSD メーリングリスト

(訳註: ここで説明しているのは本家のメーリングリストです。日本語のメーリングリストについては 日本語メーリングリストのページを見てください。)

NetBSD プロジェクトでは、 NetBSD ユーザーと開発者が容易にやりとりできるよう、 さまざまな公開メーリングリストを用意しています。 メーリングリストはすべて、メーリングリスト管理パッケージ Majordomo でサービスされており、 majordomo@NetBSD.org に空のメールを出すと、 NetBSD プロジェクトの Majordomo の使い方がわかるようになっています。

このページでは、各メーリングリストについて説明しているほか、 メーリングリストのアーカイブの閲覧、 さらにはメーリングリストの講読手続きもできるようになっています。 このページは、三つの部分に分かれています: この導入部分; すぐに見つけられるようメーリングリストの索引; そして、各メーリングリストの説明 ─ここにはアーカイブと講読手続へのリンクもあります。

メーリングリストを講読するには、このページの説明の部分にある リスト名の後のリンク をクリックします。 それからフォームに e-mail アドレスを記入してください。

メーリングリストから脱退するには、講読手続きに使ったアドレスからmajordomo@NetBSD.org へ、本文に unsubscribe listname とだけ書いたメールを送ります。

メーリングリストのアーカイブを見るには、このページの説明の部分にある リスト名の後のリンク をクリックします。これでメールアーカイブサーバー mail-index.NetBSD.org を利用できます (ネットワークアクセスとこのマシンのラックスペースを提供してくださった ISC と、 サイトの構築と保守をしてくださった Tim Rightnour のおかげです) 。 このほか、メーリングリストのアーカイブは、 Geocrawler にもあります。

メーリングリストをニュースフロントエンドで読むには、 このページの説明の部分にあるリスト名の後のリンク をクリックします。これで news.NetBSD.org のニュースアーカイブを利用できます。

NetBSD のメーリングリストの多くは、普通の unmoderated なものですが、 管理者によって moderated にされるような状況がたまに起きることに ご注意ください。

リストの説明

全般のリスト

current-users:
NetBSD-current ソースツリーの最新バージョンを使っている すべてのユーザーのための公開討論の場です。ソースツリー中のバグ等について 語るのによい場所です。 NetBSD-current ツリーは常に変化しているので、 投稿前に最新のツリーを確認するようにしてください。
netbsd-advocacy:
プロジェクトの進め方についての公開討論のためのメーリングリストです。
netbsd-announce:
NetBSD に関するお知らせのためのメーリングリストです。
netbsd-bugs:
NetBSD のバグに関する議論のためのメーリングリストです。 send-pr(1) で送られた NetBSD のバグレポートはすべてここに流れます。
pkgsrc-bugs:
pkgsrc のバグに関する議論のためのメーリングリストです。 send-pr(1) で送られた 'pkg' カテゴリーのバグレポートはすべてここに流れます。
netbsd-docs:
NetBSD の文書に関する議論のためのメーリングリストです。翻訳者用メーリングリスト に関する情報はこのページにあります。
netbsd-help:
ユーザーが質問を寄せる一般的な場としてのメーリングリストです。
netbsd-java:
NetBSD 上の Java に関連することがらの公開討論の場です。
netbsd-jobs:
NetBSD ユーザーや開発者と関連のある求人広告のための メーリングリストです。
netbsd-news:
NetBSD に関するニュースのためのメーリングリストです。 サードパーティーのニュースも含みます。
netbsd-ports:
NetBSD の移植作業をしている人や興味がある人のためのメーリングリストです。
netbsd-users:
すべての NetBSD ユーザー向けの、プラットフォームによらない一般的な 公開討論の場です。おおぜいの参加者がこのリストを読んでいるため、 よく考えた上で投稿してください。 NetBSD 自体の保守や設計に関する 疑問やコメントは、 NetBSD の使用に関する話題とは異なるため、 このリストにはふさわしくありません。そのような話題は、 狭い範囲に特化した技術的なリストがふさわしいとされています。
pkgsrc-bulk:
pkgsrc バルク構築の報告と構築ログが流れるメーリングリストです。
pkgsrc-changes:
NetBSD パッケージコレクション (pkgsrc) のすべての変更についての commit メッセージが流れるメーリングリストです。 毎日、 24 時間分のすべての commit メッセージをまとめて送る、ダイジェスト 版もあります。
source-changes:
NetBSD ソースツリーのすべての変更についての commit メッセージが 流れるメーリングリストです。 毎日、 24 時間分のすべての commit メッセージをまとめて送る、ダイジェスト 版もあります。
regional-au:
オーストラリア在住のユーザー向けの議論用です。
regional-boston:
ボストン在住のユーザー向けの議論用です。
regional-ca:
カナダ在住のユーザー向けの議論用です。
regional-cs:
チェコ共和国とスロバキア共和国在住のユーザー向けの議論用です。
regional-de:
ドイツおよびドイツ語圏域在住のユーザー向けの議論用です。
regional-fr:
フランス在住のユーザー向けの議論用です。
regional-it:
イタリア在住のユーザー向けの議論用です。
regional-jp:
日本在住のユーザー向けの議論用です。
regional-london:
ロンドン在住のユーザー向けの議論用です。
regional-nl:
オランダ在住のユーザー向けの議論用です。
regional-nordic:
北欧在住のユーザー向けの議論用です。
regional-nyc:
ニューヨークシティー在住のユーザー向けの議論用です。
regional-pl:
ポーランド在住のユーザー向けの議論用です。
regional-pt:
ポルトガルおよびポルトガル語圏域在住のユーザー向けの議論用です。
regional-sfba:
サンフランシスコ湾岸在住のユーザー向けの議論用です。
tech-cluster:
NetBSD ホストによるクラスターの構築や運用に関する技術的な議論用です。
tech-embed:
組み込みシステム上の NetBSD の使用に関する議論用です。
tech-install:
NetBSD のインストールソフトウェアの改良についての技術的事項の議論用です。 注意: このメーリングリストは NetBSD のインストールの手助け用ではありません。 NetBSD のインストールで問題がある場合は、ここではなく netbsd-help を見てください。
tech-kern:
カーネル全般に関する技術的事項の議論用です。
tech-misc:
他の tech-* リストのいずれにも該当しない技術的事項の議論用です。
tech-net:
NetBSD のネットワークサブシステムに関する技術的事項の議論用です。
tech-perform:
NetBSD のパフォーマンス、特にパフォーマンスの改良に関する議論用です。
tech-pkg:
NetBSD パッケージシステムに関する技術的事項の議論用です。
tech-ports:
NetBSD のさまざまなハードウェアへの移植に関する技術的事項の議論用です。
tech-security:
NetBSD のセキュリティーに関する技術的な議論用です。
tech-crypto:
NetBSD システムにおける暗号化に関する技術的な議論用です。このリストでは、 米国向け (cryptosrc-us) と国際版 (cryptosrc-intl) のいずれの暗号化についても 扱います。
tech-smp:
NetBSD のマルチプロセッサー対応に関する技術的事項の議論用です。
tech-toolchain:
toolchain に関する技術的な議論用です。
tech-userlevel:
POSIX 準拠、コンパイル環境など、さまざまな カーネル以外に関することの技術的事項の議論用です。
tech-x11:
NetBSD で使われる X11 ウィンドウシステムに関する議論用です。
www-changes:
NetBSD WWW サーバーのすべての変更についての commit メッセージが流れるメーリングリストです。

Broadcom の SiByte Broadband Processor ベースのプラットフォーム上の NetBSD (NetBSD/sbmips) 固有の事項の議論用です。 MIPS ベースのすべてのポートに関する質問のための port-mips リストもあります。
port-sgimips: (homepage)
SGI の MIPS プラットフォーム上の NetBSD (NetBSD/sgimips) 固有の事項の議論用です。 MIPS ベースのすべてのポートに関する質問のための port-mips リストもあります。
port-sh3: (homepage)
日立 Super-H (SH3/4) チップベースの NetBSD ポートすべてに関する事項の 議論用です。 ポート別のリストがありますので、各ポートに固有の質問や情報は そちらをご利用ください: port-dreamcast、 port-hpcsh。
port-sh5: (homepage)
SH5 ベースの NetBSD ポートすべてに関する事項の議論用です。
port-sparc: (homepage)
Sun の 32 ビット Sparc ベースのマシン上の NetBSD (NetBSD/sparc) 固有の事項の議論用です。
port-sparc64: (homepage)
Sun の 64 ビット Ultrasparc ベースのマシン上の NetBSD (NetBSD/sparc64) 固有の事項の議論用です。
port-sun2: (homepage)
Sun の Motorola 68010 ベースのマシン上の NetBSD (NetBSD/sun2) 固有の事項の議論用です。より新しい Sun のマシンは、 port-sun3、 port-sparc、 port-sparc64 がカバーします。 Motorola 68k ベースのすべてのポートに関する質問のための port-m68k リストもあります。
port-sun3: (homepage)
Sun の Motorola 68020 (sun3) および 68030 (sun3x) ベースのマシン上の NetBSD (NetBSD/sun3) 固有の事項の議論用です。 より新しい Sparc ベースの Sun のマシンは、 port-sparc と port-sparc64 がカバーします。 Motorola 68k ベースのすべてのポートに関する質問のための port-m68k リストもあります。
port-vax: (homepage)
Digital の VAX シリーズのマシン上の NetBSD (NetBSD/vax) 固有の事項の議論用です。
port-x68k: (homepage)
Sharp 製の日本の X68000/X68030 マシン上の NetBSD (NetBSD/x68k) 固有の事項の議論用です。 Motorola 68k ベースのすべてのポートに関する質問のための port-m68k リストもあります。
port-xen: (homepage)
Xen 仮想マシンモニター 上の NetBSD (NetBSD/xen) 固有の事項の議論用です。
