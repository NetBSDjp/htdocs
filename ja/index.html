Welcome to the NetBSD Project
``Of course it runs NetBSD.''

The NetBSD Project は、多くの有志の熱意により、 無料で、且つ自由に再配布できる Unix-like なオペレーティングシステムの開発を行っています。

NetBSD は多くの ハードウェア環境で作動すると 同時に、高い移植性を備えています。 また、全てのソースコードが付属しており、ユーザーにより サポートされています。
